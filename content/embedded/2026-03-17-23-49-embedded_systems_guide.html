一句话理解

你的手机、智能手表、扫地机器人、汽车ABS刹车系统、微波炉、蓝牙耳机……里面都藏着一个嵌入式系统。

它不像电脑那样什么都能干，而是专为特定功能设计的专用计算机。

智能洗衣机

控制洗涤程序

扫地机器人

路径规划+避障

汽车ECU

发动机控制

血糖仪

精准测量+显示

三大核心特点

专用性

只干一件事，不跑其他软件

洗衣机只控制洗涤程序，不能刷抖音

资源受限

CPU慢、内存小、功耗低

智能手环电池能用一周，手机一天就没电

实时响应

必须在规定时间内完成任务

汽车安全气囊必须在碰撞瞬间弹出

零基础学习路线 三个阶段，从入门到进阶

第一阶段

打基础

2-3周 · 理论根基

  • C语言：精通指针、结构体、位操作
  • 电路基础：认识电阻电容，看懂GPIO
基础代码示例 
// GPIO控制LED
#define LED_PIN 13

void setup() {
    pinMode(LED_PIN, OUTPUT);
}

void loop() {
    digitalWrite(LED_PIN, HIGH);
    delay(1000);
    digitalWrite(LED_PIN, LOW);
    delay(1000);
}
