🖥️ 第一章：认识你的电脑

1.1 电脑是什么

电脑（计算机）就像一个超级聪明的电子助手，它可以帮助我们：

  • 📝 处理文档 - 写文章、做表格、制作PPT
  • 🌐 上网浏览 - 看新闻、看视频、和朋友聊天
  • 🎵 娱乐休闲 - 听音乐、看电影、玩游戏
  • 📸 存储回忆 - 保存照片、视频和重要文件
  • 💼 办公学习 - 远程办公、在线学习、查资料

1.2 电脑的主要部件

🖥️
显示器
像电视屏幕，显示内容
⌨️
键盘
输入文字和数字
🖱️
鼠标
控制光标，点击选择
🎛️
主机/机身
电脑的大脑和心脏

台式电脑 vs 笔记本电脑

🖥️ 台式电脑

  • 显示器、主机、键盘鼠标分开
  • 屏幕大，适合固定场所使用
  • 性能更强，价格更实惠
  • 需要连接电源使用

💻 笔记本电脑

  • 所有部件集成在一起
  • 体积小，方便携带
  • 自带电池，可移动使用
  • 价格相对较高

1.3 开机和关机

📟 开机步骤
  1. 确保电源线已连接
  2. 按下主机或笔记本上的 电源按钮（通常有 💡 或 ⭘ 符号）
  3. 等待屏幕亮起，出现 Windows 或 Mac 标志
  4. 输入密码（如果设置了）进入桌面
🛑 正确关机步骤（非常重要！）
  1. 点击屏幕左下角的 开始菜单（Windows 图标 🪟）
  2. 点击 电源 图标
  3. 选择 关机
  4. 等待屏幕完全变黑，再关闭显示器电源
🚫 千万不要这样做！
  • 不要直接拔掉电源线关机（会损坏电脑和数据）
  • 不要长按电源键强制关机（除非死机无响应）
  • 不要在开机状态下移动台式电脑

🪟 第二章：操作系统基础

2.1 什么是操作系统

操作系统是电脑的大管家，负责管理所有硬件和软件。常见的有：

  • Windows - 微软出品，全球使用最广泛（本指南主要介绍）
  • macOS - 苹果电脑专用，界面简洁美观
  • Linux - 开源免费，技术人员常用

2.2 认识 Windows 桌面

🖥️ 桌面主要元素

  • 桌面背景 - 可以更换成喜欢的图片
  • 图标 - 程序、文件、文件夹的快捷方式
  • 任务栏 - 底部的一长条，显示正在运行的程序
  • 开始菜单 - 左下角的 Windows 图标，打开所有程序
  • 系统托盘 - 任务栏右下角，显示时间、网络、音量等

2.3 任务栏详解

位置 功能
左侧 Windows 图标 打开开始菜单，访问所有程序
中间图标 固定的程序或打开的程序窗口
右侧小图标 输入法、网络、音量、电池状态
最右侧时间 显示日期时间，点击可查看日历

2.4 开始菜单的使用

点击左下角 Windows 图标 🪟 或按键盘上的 Win 键打开开始菜单：

  • 已固定 - 常用的程序快捷方式
  • 所有应用 - 按字母排序的所有程序列表
  • 推荐的项目 - 最近打开的文件
  • 搜索框 - 输入程序名称快速查找
  • 用户头像 - 切换用户、锁定、关机
💡 快速找程序 不知道程序在哪？直接按 Win 键，然后输入程序名称（如"微信"、"记事本"），按 Enter 即可打开！

📁 第三章：文件管理 - 让你的资料井井有条

3.1 文件和文件夹的概念

🌰 用衣柜来理解

  • 文件 = 具体的衣服（衬衫、裤子）
  • 文件夹 = 抽屉或收纳盒（放同类衣服的容器）
  • 磁盘分区 = 衣柜的不同区域（C盘、D盘）

3.2 打开文件资源管理器

有多种方法打开（选一种喜欢的）：

  1. 按 Win + E 快捷键
  2. 右键点击开始菜单 → 选择"文件资源管理器"
  3. 点击任务栏上的文件夹图标 📁

3.3 文件资源管理器界面

📂 左侧导航栏

快速访问、此电脑、网络位置等

📄 中间内容区

显示当前文件夹内的所有内容

🛤️ 地址栏

显示当前位置路径

🔍 搜索框

在当前位置搜索文件

3.4 认识磁盘分区

打开"此电脑"，你会看到类似 C:、D: 的盘符：

盘符 用途建议 说明
C: 盘 系统和程序 安装 Windows 的地方，不要乱放文件
D: 盘 个人文件 存放文档、照片、视频等
E: 盘 备份/大文件 存放游戏、电影等大文件
⚠️ 重要提醒
  • C 盘满了会导致电脑变慢甚至无法开机
  • 重要文件不要只存在 C 盘，建议存 D 盘或云端
  • 桌面其实也在 C 盘，所以桌面不宜放太多大文件

3.5 创建文件夹整理文件

  1. 选择存放位置

    打开 D 盘（或其他位置），想好要创建什么分类的文件夹

  2. 创建新文件夹

    在空白处右键 → 新建 → 文件夹；或点击顶部"新建文件夹"按钮

  3. 命名文件夹

    输入有意义的名称，如"2024年照片"、"工作文档"、"学习资料"

  4. 移动文件

    选中文件 → 拖动到文件夹；或右键→剪切，到目标文件夹右键→粘贴

3.6 文件命名技巧

✅ 好的命名示例
  • 2024-03-09_季度总结报告.docx（日期+内容）
  • 三亚旅游_2024春节_精选照片.jpg（事件+时间）
  • 合同_张三_2024_v2.pdf（类型+人+版本）
❌ 避免这样命名
  • 新建文本文档.txt（没有实际意义）
  • 新建文件夹(2)（难以辨认）
  • aaaaaaaa.jpg（无法搜索）
  • 文件名太长（超过50个字符）

3.7 常用文件操作

操作 鼠标操作 快捷键

粘贴 右键 → 粘贴 Ctrl + V

3.8 快速搜索文件

找不到文件时，用搜索功能：

  1. 打开文件资源管理器（Win + E）
  2. 点击右上角的搜索框
  3. 输入文件名或部分关键词
  4. 等待搜索结果
💡 搜索技巧
  • 记得文件类型可以搜 *.jpg（所有图片）、*.docx（所有Word文档）
  • 记得部分内容可以搜索文档内的文字
  • 按修改日期筛选，缩小范围

🖥️ 第四章：常用软件使用指南

4.1 软件安装与卸载

安装软件

  1. 下载安装包

    从官网下载（避免下载站，可能有捆绑软件）

  2. 运行安装程序

    双击 .exe 文件，按提示下一步

  3. 注意安装选项

    ❗ 取消勾选"安装XX助手""设为XX首页"等捆绑选项

  4. 选择安装位置

    建议改到 D 盘，如 D:\Program Files\软件名

🚫 避坑指南 - 避免"全家桶"

国内某些下载站会捆绑安装其他软件。注意：

  • 只从官网下载软件
  • 安装时看清每一步，取消勾选额外软件
  • 不要点击"高速下载"（通常会下载下载器）
  • 选择"普通下载"或"本地下载"

卸载软件

  1. 按 Win + I 打开设置
  2. 选择"应用" → "应用和功能"
  3. 找到要卸载的软件
  4. 点击"卸载"按钮

5.5 网络安全 - 非常重要！

🚨 防骗守则
  • ❌ 不要点击陌生链接，尤其是短信、QQ/微信里的
  • ❌ 不要下载来源不明的软件
  • ❌ 不要在非官方网站输入密码
  • ❌ 不要轻信"中奖""退款""安全账户"等话术
  • ❌ 不要向任何人透露验证码
🔒 账号安全建议
  • 使用复杂密码（字母+数字+符号，8位以上）
  • 重要账号开启双重验证（短信或邮箱验证）
  • 不同网站使用不同密码（可用密码管理器）
  • 定期更换重要账号密码
  • 安装杀毒软件，定期扫描

5.6 识别钓鱼网站

⚠️ 可疑特征 ✅ 正确做法
网址拼写错误（如 ta0bao.com） 仔细核对网址，手动输入常用网站
要求输入银行卡密码 正规网站不会索要银行卡密码
弹窗提示"电脑中毒" 关闭页面，用正版杀毒软件扫描
网址是 http 而非 https 敏感操作确保网址以 https:// 开头

🔧 第六章：电脑维护与优化

6.1 让电脑保持流畅

🗑️ 定期清理垃圾

使用磁盘清理工具或 CCleaner 清理临时文件

🚀 关闭开机启动项

任务管理器 → 启动 → 禁用不需要的程序

💾 清理桌面

桌面文件太多会拖慢系统，移到 D 盘

🦠 定期杀毒

每周全盘扫描一次，保持杀毒软件更新

6.2 磁盘清理步骤

  1. 打开"此电脑"，右键 C 盘 → 属性
  2. 点击"磁盘清理"
  3. 勾选要删除的文件类型（临时文件、回收站等）
  4. 点击"确定"清理

6.3 常见问题解决

问题 解决方法
电脑卡死/无响应 按 Ctrl + Alt + Delete → 任务管理器 → 结束无响应程序
无法上网 检查 Wi-Fi 是否连接；重启路由器；运行网络诊断
声音没有了 检查音量图标是否静音；检查音箱/耳机连接；更新声卡驱动
软件打不开 重启电脑；重新安装软件；检查是否被杀毒软件拦截
蓝屏死机 记录错误代码；重启；如频繁出现需检查硬件或重装系统
C 盘空间不足 磁盘清理；卸载不用的软件；移动文件到 D 盘

6.4 重要数据备份

💾 备份建议

重要文件要遵循"3-2-1"备份原则：

  • 3 份副本（1份原件 + 2份备份）
  • 2 种不同存储介质（如电脑 + U盘/移动硬盘）
  • 1 份存放在异地（如云盘）

备份方案推荐

  1. 本地备份 - 重要文件定期复制到移动硬盘或 U 盘
  2. 云盘备份 - 使用百度网盘、阿里云盘等自动同步重要文件夹
  3. 微信文件传输助手 - 临时保存小文件到手机
