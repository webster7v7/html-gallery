💚 Vue 3 核心概念

📦 ref - 响应式数据

ref 是 Vue 3 的核心，用来定义响应式数据。当数据变化时，页面会自动更新！

<script setup lang="ts">
import { ref } from 'vue'

// ========== 定义响应式数据 ==========
const count = ref(0)           // 数字
const name = ref('张三')        // 字符串
const list = ref([1, 2, 3])    // 数组
const user = ref({ age: 20 })  // 对象

// 读取/修改 ref 数据要用 .value
console.log(count.value)       // 0
count.value = 10               // 修改后页面自动更新

const increment = () => {
  count.value++
}
</script>

<template>
  <!-- 模板中直接使用，不需要 .value -->
  <p>计数: {{ count }}</p>
  <button @click="increment">+1</button>
</template>
💡 关键概念：为什么用 ref 而不是普通变量？Vue 需要知道数据何时变化来更新页面，ref 包装后的数据变化时，Vue 会自动刷新相关页面部分。

🔄 组件通信

父传子 (Props)：

<!-- 子组件 -->
<script setup lang="ts">
interface Props {
  title: string
  count?: number  // ? 表示可选
}
const props = defineProps<Props>()
</script>

<template>
  <div>{{ title }} - {{ count }}</div>
</template>

子传父 (Emits)：

<!-- 子组件 -->
<script setup lang="ts">
const emit = defineEmits<{
  (e: 'update', value: number): void
  (e: 'delete', id: number): void
}>()

const handleClick = () => {
  emit('update', 100)
}
</script>

📝 表单双向绑定

<script setup lang="ts">
import { ref } from 'vue'

const username = ref('')
const isChecked = ref(false)
const selected = ref('a')
</script>

<template>
  <!-- 文本输入 -->
  <input v-model="username" placeholder="用户名" />
  
  <!-- 复选框 -->
  <input type="checkbox" v-model="isChecked" />
  
  <!-- 单选下拉 -->
  <select v-model="selected">
    <option value="a">选项 A</option>
    <option value="b">选项 B</option>
  </select>
</template>

// 基本类型

  return a + b
}

🔷 泛型 (Generic)

泛型让代码更灵活，可以处理多种类型。

// 泛型接口
interface ApiResponse<T> {
  data: T
  code: number
  message: string
}

// 使用
interface User {
  id: number
  name: string
}

const response: ApiResponse<User> = {
  data: { id: 1, name: '张三' },
  code: 200,
  message: 'success'
}

🌐 Axios 使用指南

🔵 基础用法

import axios from 'axios'

// GET 请求
const getUsers = async () => {
  const response = await axios.get('https://api.example.com/users')
  return response.data
}

// POST 请求
const createUser = async (userData: any) => {
  const response = await axios.post('https://api.example.com/users', userData)
  return response.data
}

// 带参数
const searchUsers = async (keyword: string) => {
  const response = await axios.get('/users', {
    params: { keyword, page: 1 }
  })
  return response.data
}

🔧 封装请求（最佳实践）

// utils/request.ts
import axios from 'axios'

const request = axios.create({
  baseURL: 'https://api.example.com',
  timeout: 10000,
  headers: {
    'Content-Type': 'application/json'
  }
})

// 请求拦截器
request.interceptors.request.use(
  (config) => {
    // 添加 token
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  (error) => Promise.reject(error)
)

// 响应拦截器
request.interceptors.response.use(
  (response) => response.data,
  (error) => {
    // 统一错误处理
    if (error.response?.status === 401) {
      // 未授权，跳转登录
    }
    return Promise.reject(error)
  }
)

export default request

❓ 常见问题

Q: 为什么用 ref 而不是普通变量？

A: Vue 需要知道数据何时变化来更新页面，ref 包装后的数据变化时，Vue 会自动刷新相关页面部分。

Q: .value 是什么？

A: ref 包装的数据需要通过 .value 来读取和修改。但在模板中不需要写 .value。

Q: Tailwind 类名太多记不住怎么办？

A: 安装 VS Code 的 Tailwind 插件，输入时会自动提示。或者查阅官方文档的搜索功能。

Q: TypeScript 报错怎么办？

A: 新手可以先写普通 JS，等运行正常后再添加类型。也可以将 tsconfig.json 中的 "strict": true 改为 false 降低严格度。
