🎬 达芬奇是什么？

DaVinci Resolve（达芬奇）是 Blackmagic Design 公司开发的一款专业级视频后期制作软件，集剪辑、调色、特效、音频后期于一体。它是好莱坞电影后期制作的标准工具之一，而免费版就已经足够大多数用户使用！

✂️ 剪辑（Edit）

专业的非线性剪辑功能，支持多轨道剪辑、智能剪辑工具、多机位编辑

🎨 调色（Color）

业界最强大的调色系统，一级调色、二级调色、LUT、HDR 支持一应俱全

✨ 特效（Fusion）

节点式合成工具，制作视觉特效、动态图形、字幕动画

🎵 音频（Fairlight）

专业音频后期页面，混音、降噪、音频修复功能强大

📤 交付（Deliver）

多种格式导出，支持 YouTube、B站、抖音等平台预设

🆓 完全免费

免费版功能已经非常强大，付费版（Studio）主要是多用户协作和部分高级功能

💡 免费版 vs 付费版

  • 免费版：适合个人用户，4K 以下分辨率，无水印，功能已经足够强大
  • Studio 版（¥2550）：支持 8K+、降噪、HDR 调色、多用户协作、AI 功能等
  • 新手建议先用免费版，觉得不够用了再考虑升级

🖥️ 界面认识

达芬奇有 7 个主要页面，通过底部导航栏切换：

🏠 Media（媒体页面）

导入素材、预览素材、管理媒体文件。可以在这里浏览硬盘，选择要导入的素材。

✂️ Cut（快编页面）

快速剪辑页面，适合粗剪和快速筛选素材。界面简洁，适合快速拼接镜头。

🎬 Edit（剪辑页面）

主要剪辑工作区，功能最全面。多轨道时间线、转场、字幕、特效都在这。

🎨 Fusion（特效页面）

节点式合成工具，制作视觉特效、动态图形、复杂的合成效果。

🌈 Color（调色页面）

达芬奇的看家本领，业界最强大的调色工具。一级调色、二级调色、LUT、HDR。

🎵 Fairlight（音频页面）

专业音频后期页面，混音台、均衡器、降噪、音频修复工具。

📤 Deliver（交付页面）

导出渲染页面，设置输出格式、分辨率、编码，渲染最终视频。

💡 新手建议

  • 刚开始主要使用 Edit（剪辑）、Color（调色）、Deliver（导出） 三个页面
  • Cut 页面适合快速粗剪，Edit 页面适合精剪
  • 按 Shift + 2~8 可以快速切换页面

Edit 页面布局

区域 位置 功能

预览窗口 中上 预览选中的素材或时间线画面

🔄 标准工作流程

1. 新建项目 → 2. 导入素材 → 3. 粗剪 → 4. 精剪 → 5. 调色 → 6. 音频 → 7. 导出

详细步骤

  • 新建项目

    打开达芬奇 → 点击 "New Project" → 输入项目名称 → 点击 "Create"

  • 导入素材

    切换到 Media 页面 → 在 Media Storage 中找到素材文件夹 → 选中素材右键 "Add to Media Pool"，或直接拖入 Media Pool

  • 创建时间线

    切换到 Edit 页面 → Media Pool 中选中素材 → 右键 "Create New Timeline Using Selected Clips"，或手动拖入时间线

  • 粗剪（Cut 页面）

    切换到 Cut 页面 → 快速筛选素材，保留需要的片段，删除废片，确定大致顺序

  • 精剪（Edit 页面）

    回到 Edit 页面 → 精细调整剪辑点 → 添加转场 → 添加字幕 → 调整音频电平

  • 调色（Color 页面）

    切换到 Color 页面 → 一级调色（白平衡、曝光、对比度）→ 二级调色（HSL 选区、跟踪）→ 添加风格化 LUT

  • 音频处理（Fairlight 页面）

    切换到 Fairlight 页面 → 降噪 → 均衡器调整 → 混音 → 添加背景音乐

  • 导出（Deliver 页面）

    切换到 Deliver 页面 → 选择预设或自定义设置 → 添加到渲染队列 → 开始渲染

时间线缩放与导航

💡 高效操作技巧

  • 按住 Ctrl + 滚轮：缩放时间线
  • 按住 Shift + 滚轮：横向滚动时间线
  • 按 \（反斜杠）：自动缩放适应整个时间线
  • 按 D：启用/禁用片段（不删除但暂时不显示）
  • 按 L 多次：倍速播放（正向），J：倒放

添加转场

  • 打开特效库

    点击左上角 "Effects Library" 按钮，或按 Shift+2

  • 选择转场

    展开 "Toolbox" → "Video Transitions"，选择喜欢的转场（推荐 Cross Dissolve 叠化）

  • 应用转场

    将转场拖放到两个片段的交界处

  • 调整时长

    选中转场，在检查器中调整 Duration（时长），建议 0.5-1 秒

添加字幕

  • 打开字幕工具

    Effects Library → "Titles" → 选择 "Text" 或 "Text+"

  • 放置字幕

    拖到时间线上，放在单独的轨道（建议 V2 或更高）

  • 编辑文字

    选中字幕 → 检查器中修改文字内容、字体、大小、颜色

  • 调整位置时长

    拖动字幕边缘调整时长，预览窗口拖动调整位置

⚠️ 剪辑注意事项

  • 定期保存项目：虽然达芬奇有自动保存，但重要节点建议手动 Ctrl+S
  • 建立素材管理习惯：按类型分 Bin（文件夹）管理素材
  • 剪辑前先代理：4K 素材卡顿时，生成代理文件再剪辑

🎨 调色基础

调色是达芬奇的强项，即使是免费版也有业界顶尖的调色工具。

调色页面布局

区域 位置 功能
画廊 左上 保存和调用调色预设、静帧
节点编辑器 右上 显示当前节点的处理流程
检视器 中间 预览画面，可对比调色前后
时间线 中下 选择要调色的片段
调色面板 左下 色轮、曲线、色条等调色工具
关键帧 右下 设置动态调色关键帧

一级调色（整体调整）

先进行一级调色，校正画面的基本影调：

⚪ 白平衡

用吸管工具点击画面中应该是白色的地方，自动校正色温

☀️ 曝光

调整 Lift（暗部）、Gamma（中间调）、Gain（高光），让画面亮度适中

🌓 对比度

调整 Contrast，或分别调整 Lift 和 Gain 来增强画面对比

🎨 饱和度

调整 Saturation，让画面色彩更加鲜艳或柔和

二级调色（局部调整）

用限定器（Qualifier）选择特定颜色进行调整：

  • 打开限定器

    在调色面板点击 "Qualifier"（吸管图标）

  • 选择颜色

    用吸管在画面上点击要调整的颜色（如天空的蓝色）

  • 调整选区

    调整 3D/HSL 参数，让选区更精确（可以开启高亮显示查看选区范围）

  • 调整颜色

    用色轮或色条调整选中区域的颜色、饱和度、亮度

使用 LUT

LUT（查找表）是快速调色的好方法：

  • 应用 LUT

    右键点击节点 → "LUTs" → 选择预设的 LUT，或加载外部 .cube 文件

  • 调整强度

    在 Key 面板调整 "Key Output Gain" 来控制 LUT 强度（0-1）

  • 推荐 LUT 网站

    Ground Control Color、Phantom LUTs、Moment Cine LUTs

💡 调色小技巧

  • 开启 "Highlight"（高亮）查看曝光是否过曝
  • 用 "Before/After" 对比调色前后效果
  • 调色前先统一所有镜头的白平衡和曝光
  • 复制调色：选中已调色的片段 → 右键 "Grab Still" → 应用到其他片段

📤 导出设置

导出步骤

  • 切换到 Deliver 页面

    点击底部导航栏 "Deliver" 或按 Shift+8

  • 选择渲染范围

    选择 "Entire Timeline"（整个时间线）或 "In/Out Range"（入点出点之间的范围）

  • 选择预设或自定义

    右侧选择预设（YouTube、Vimeo 等），或自定义设置

  • 设置输出路径

    点击 "Location" 选择保存位置，命名文件

  • 添加到渲染队列

    点击 "Add to Render Queue"，右侧队列会显示任务

  • 开始渲染

    点击 "Render All" 开始导出

推荐导出设置

📺 YouTube/B站（推荐）

  • 格式：MP4
  • 编解码器：H.264 或 H.265（HEVC）
  • 分辨率：1920×1080 或 3840×2160
  • 帧率：与素材一致
  • 码率：20-50 Mbps（1080p）

📱 抖音/快手（竖屏）

  • 格式：MP4
  • 编解码器：H.264
  • 分辨率：1080×1920
  • 帧率：30fps 或 60fps
  • 码率：8-15 Mbps

🎬 高质量存档

  • 格式：QuickTime
  • 编解码器：ProRes 422 HQ 或 DNxHR
  • 分辨率：原始分辨率
  • 帧率：原始帧率
  • 注意：文件会很大

💻 网络预览

  • 格式：MP4
  • 编解码器：H.264
  • 分辨率：1280×720
  • 帧率：24/25/30fps
  • 码率：5-8 Mbps

⚠️ 导出注意事项

  • 导出前务必检查时间线，确认没有片段离线（显示红色）
  • 分辨率建议保持原始素材分辨率，不要放大
  • H.265（HEVC）比 H.264 体积小一半，但兼容性稍差
  • 导出时可以做其他事，但不要关闭达芬奇或休眠电脑
