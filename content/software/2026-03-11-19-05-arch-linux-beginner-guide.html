0准备阶段：你需要什么？

1. 下载镜像

  • 访问 archlinux.org 下载 ISO 文件（约 800MB）
  • 用 Rufus（Windows）或 balenaEtcher 制作 U 盘启动盘

2. 检查启动模式

重要：开机按 F2/F12/Del 进 BIOS，关闭 Secure Boot，确认是 UEFI 模式（2015年后电脑一般都支持）

3. 网络连接

Arch 安装需要联网下载软件。建议插网线，WiFi 也可以在安装界面配置。

1安装 Arch Linux（核心步骤）

以下命令按顺序输入，Enter 执行。以 # 开头的是注释，不用输入。

步骤 1：验证启动模式

[root@archiso ~]# ls /sys/firmware/efi/efivars

如果显示一堆文件，说明是 UEFI 模式，继续。如果报错，检查 BIOS 设置。

步骤 2：连接网络

[root@archiso ~]# dhcpcd
[root@archiso ~]# iwctl
[iwd]# device list  # 查看网卡名，如 wlan0
[iwd]# station wlan0 scan
[iwd]# station wlan0 get-networks  # 显示 WiFi 列表
[iwd]# station wlan0 connect "你的WiFi名"
[iwd]# exit

步骤 3：更新系统时钟

[root@archiso ~]# timedatectl set-ntp true

步骤 4：硬盘分区（⚠️ 小心操作）

警告：这会清空硬盘数据！双系统用户请谨慎，建议先备份。
[root@archiso ~]# fdisk -l

找到你的硬盘，通常是 /dev/sda 或 /dev/nvme0n1（NVMe 固态）。

[root@archiso ~]# cfdisk /dev/sda  # 用你的硬盘名替换

在 cfdisk 界面中：

  1. 选择 New → 输入 512M → 类型选 EFI System（启动分区）
  2. 再选 New → 输入剩余全部空间（直接回车）→ 类型选 Linux filesystem
  3. 选 Write 写入，输入 yes 确认
  4. 选 Quit 退出

步骤 5：格式化分区

# 格式化 EFI 分区（假设是 /dev/sda1）
[root@archiso ~]# mkfs.fat -F32 /dev/sda1

# 格式化根分区（假设是 /dev/sda2）
[root@archiso ~]# mkfs.ext4 /dev/sda2

步骤 6：挂载分区

[root@archiso ~]# mount /dev/sda2 /mnt
[root@archiso ~]# mkdir -p /mnt/boot/efi
[root@archiso ~]# mount /dev/sda1 /mnt/boot/efi

步骤 7：安装基础系统

[root@archiso ~]# pacstrap /mnt base linux linux-firmware base-devel vim nano sudo

等待 5-15 分钟，取决于网速。这是安装系统核心文件。

步骤 8：生成配置文件

[root@archiso ~]# genfstab -U /mnt >> /mnt/etc/fstab

步骤 9：进入新系统

[root@archiso ~]# arch-chroot /mnt

现在你已经进入安装好的 Arch 系统里了，提示符会变成 [root@archiso /]#

步骤 10：设置时区和语言

# 设置时区为上海
[root@archiso /]# ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
[root@archiso /]# hwclock --systohc

# 设置语言（编辑配置文件）
[root@archiso /]# nano /etc/locale.gen

在 nano 编辑器中：

  • 找到 #zh_CN.UTF-8 UTF-8 和 #en_US.UTF-8 UTF-8
  • 删除前面的 # 号（取消注释）
  • 按 Ctrl+O 保存，Ctrl+X 退出
[root@archiso /]# locale-gen
[root@archiso /]# echo "LANG=zh_CN.UTF-8" > /etc/locale.conf

步骤 11：设置主机名和 hosts

[root@archiso /]# echo "myarch" > /etc/hostname

[root@archiso /]# nano /etc/hosts

在 hosts 文件中添加以下三行：

127.0.0.1   localhost
::1         localhost
127.0.1.1   myarch.localdomain   myarch

步骤 12：设置 root 密码和创建用户

# 设置 root 密码（输入时不会显示字符）
[root@archiso /]# passwd

# 创建普通用户（替换 webster 为你想要的用户名）
[root@archiso /]# useradd -m -G wheel -s /bin/bash webster
[root@archiso /]# passwd webster
[root@archiso /]# EDITOR=nano visudo

找到 # %wheel ALL=(ALL:ALL) ALL 这一行，删除 # 号，保存退出。

步骤 13：安装引导程序（关键！）

[root@archiso /]# pacman -S grub efibootmgr
[root@archiso /]# grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=GRUB
[root@archiso /]# grub-mkconfig -o /boot/grub/grub.cfg

步骤 14：退出并重启

[root@archiso /]# exit  # 退出 chroot 环境
[root@archiso ~]# umount -R /mnt  # 卸载分区
[root@archiso ~]# reboot  # 重启
重启后拔掉 U 盘！如果看到 GRUB 菜单，选择 Arch Linux，输入用户名密码登录，恭喜你安装成功！

2安装桌面环境（图形界面）

刚装好的 Arch 只有黑屏命令行，需要手动安装图形界面。推荐两种方案：

方案 A：轻量级 XFCE（适合老旧电脑）

# 安装显示服务器和桌面
[webster@myarch ~]$ sudo pacman -S xorg xfce4 xfce4-goodies lightdm lightdm-gtk-greeter

# 设置开机启动显示管理器
[webster@myarch ~]$ sudo systemctl enable lightdm

# 重启进入图形界面
[webster@myarch ~]$ sudo reboot

方案 B：美观的 KDE Plasma（推荐）

[webster@myarch ~]$ sudo pacman -S xorg plasma kde-applications sddm
[webster@myarch ~]$ sudo systemctl enable sddm
[webster@myarch ~]$ sudo reboot
安装显卡驱动：如果进桌面后分辨率不对或卡顿，需要装显卡驱动。
NVIDIA 显卡：sudo pacman -S nvidia
Intel 核显：sudo pacman -S xf86-video-intel
AMD 显卡：sudo pacman -S xf86-video-amdgpu

3安装常用软件

安装 AUR 助手（必备）

AUR 是社区软件仓库，软件比官方仓库多得多。yay 是最好的 AUR 助手。

[webster@myarch ~]$ sudo pacman -S git
[webster@myarch ~]$ git clone https://aur.archlinux.org/yay.git
[webster@myarch ~]$ cd yay
[webster@myarch yay]$ makepkg -si

小白必备软件清单

# 浏览器、输入法、终端增强
[webster@myarch ~]$ sudo pacman -S firefox fcitx5-im fcitx5-chinese-addons

# 文件管理器、压缩工具、字体
[webster@myarch ~]$ sudo pacman -S thunar file-roller noto-fonts-cjk

# 播放器、截图、编辑器
[webster@myarch ~]$ sudo pacman -S vlc flameshot code

# 微信、QQ、网易云音乐（AUR）
[webster@myarch ~]$ yay -S wechat-uos linuxqq netease-cloud-music
yay 用法：和 pacman 几乎一样。yay -S 软件名 安装，yay -R 软件名 卸载，yay 更新所有软件（包括 AUR）。

4日常使用命令（生存必备）

命令 作用 示例
sudo pacman -Syu 更新系统（每周运行一次） 升级所有软件包
sudo pacman -S 名 安装软件 sudo pacman -S firefox
sudo pacman -R 名 卸载软件 sudo pacman -R firefox
pacman -Ss 关键词 搜索软件包 pacman -Ss video player
yay -S 名 从 AUR 安装 yay -S google-chrome
neofetch 显示系统信息 截图装逼用
htop 任务管理器 看 CPU 内存占用
sudo reboot 重启 -
sudo shutdown now 关机 -

文件管理基础

ls 查看当前文件夹内容
cd 文件夹名 进入文件夹
cd .. 返回上级文件夹
pwd 显示当前路径
mkdir 名 创建文件夹
rm -rf 名 删除文件夹（慎用！）

5常见问题解决

🔴 无法启动 / 黑屏

  • 检查 BIOS 是否关闭 Secure Boot
  • NVIDIA 显卡可能需要额外参数：nomodeset
  • 在 GRUB 界面按 e 编辑启动项，在 linux 行末尾加 nomodeset，按 F10 启动

🔴 没有声音

[webster@myarch ~]$ sudo pacman -S alsa-utils pulseaudio pavucontrol
[webster@myarch ~]$ alsamixer  # 检查音量是否静音（MM 表示静音，按 M 切换）

🔴 中文显示为方块

[webster@myarch ~]$ sudo pacman -S noto-fonts-cjk wqy-zenhei

🔴 无法输入中文

需要在桌面环境设置中添加输入法：

  1. 打开 Fcitx5 配置
  2. 添加 Pinyin 或 Rime
  3. 设置环境变量（添加到 ~/.bashrc）：
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx

🔴 更新后无法启动（Kernel 问题）

# 用 Live USB 启动，进入 Arch ISO，挂载分区后 chroot
[root@archiso /]# mkinitcpio -P
[root@archiso /]# grub-mkconfig -o /boot/grub/grub.cfg
