🤔 什么是 1Panel？

1Panel 是一个现代化、开源的 Linux 服务器运维管理面板。简单来说，它就像是一个"服务器遥控器"，让你不用记住复杂的命令，通过网页点点鼠标就能管理服务器！

🖥️

可视化管理

不用记命令，网页界面操作，像用 Windows 一样简单

🐳

Docker 集成

深度集成 Docker，一键部署各种应用

🛡️

安全防护

防火墙、SSL 证书、病毒扫描，全方位保护

📊

监控面板

实时监控服务器状态，CPU、内存、磁盘一目了然

📋 1Panel 能做什么？

  • 网站管理 - 快速搭建网站，支持 PHP、Java、Python、Node.js 等多种环境
  • 数据库管理 - 一键安装 MySQL、PostgreSQL、Redis 等数据库
  • 文件管理 - 网页版文件管理器，上传下载、编辑文件超方便
  • 应用商店 - 100+ 热门应用一键安装（WordPress、Nginx、宝塔迁移工具等）
  • SSL 证书 - 自动申请和续期 HTTPS 证书
  • 计划任务 - 定时备份、自动清理，自动化运维
  • 安全审计 - 登录日志、操作记录，安全有保障

💡 小白提示

如果你之前用过宝塔面板，可以把 1Panel 理解成宝塔的"开源升级版"，界面更现代，功能更强大，而且完全免费！

📦 准备工作

在安装 1Panel 之前，你需要准备以下内容：

项目 要求 说明
🖥️ 服务器 1核2G 以上 推荐阿里云、腾讯云、华为云等
💿 系统 CentOS 7+ / Ubuntu 20+ / Debian 10+ 推荐 Ubuntu 22.04 LTS
🌐 网络 可访问互联网 用于下载和更新
🔑 权限 root 用户 需要有服务器 root 密码或密钥

⚠️ 重要提醒

1Panel 会占用服务器的 80、443、9999 等端口，请确保这些端口没有被其他程序占用。如果安装了宝塔或其他面板，建议先卸载或更换服务器。

🚀 安装 1Panel

安装非常简单，只需要在服务器上执行一条命令即可！

第一步：连接服务器

使用 SSH 工具连接到你的服务器。Windows 用户推荐用 FinalShell 或 Xshell，Mac 用户直接用终端。

🖥️

SSH 连接服务器示意图

输入命令：ssh root@你的服务器IP

第二步：执行安装命令

连接成功后，复制下面这条命令，粘贴到终端中按回车：

# 快速安装命令（国内服务器推荐） curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && sudo bash quick_start.sh # 如果是国外服务器，用这个 curl -sSL https://raw.githubusercontent.com/1Panel-dev/1Panel/main/install.sh -o install.sh && sudo bash install.sh
1

选择安装目录

安装脚本会询问安装路径，直接按 Enter 使用默认的 /opt/1panel 即可。

2

设置端口

默认使用 9999 端口，如果没有冲突直接回车。如果提示端口被占用，可以换一个，比如 8888。

3

设置安全入口

这是 1Panel 的安全特性，访问面板需要加一段随机字符串。建议保持默认或自定义一个记得住的。

例如：http://服务器IP:9999/安全入口

4

设置用户名和密码

设置登录 1Panel 的用户名和密码，一定要记好！密码输入时不会显示，这是正常的。

5

等待安装完成

安装大约需要 3-5 分钟，看到以下信息就表示安装成功了！

[SUCCESS] 1Panel 安装成功！ 面板地址：http://192.168.1.100:9999/abc123 用户名：admin 密码：你的密码 # 请保存好以上信息！

✅ 安装成功！

现在你可以用浏览器访问上面的地址，输入用户名密码登录 1Panel 了！

🛠️ 常用命令

安装完成后，在服务器上可以使用以下命令管理 1Panel：

# 查看 1Panel 状态 1pctl status # 停止 1Panel 1pctl stop # 启动 1Panel 1pctl start # 重启 1Panel 1pctl restart # 重置密码 1pctl reset password # 查看面板信息 1pctl user-info

🌐 创建你的网站

1Panel 让搭建网站变得超级简单，只需几步：

1

安装运行环境

点击左侧"应用商店"，搜索并安装 OpenResty（Nginx 的增强版，用于网站服务）

2

创建网站

点击"网站"→"创建网站"，选择"静态网站"或"反向代理"。输入域名，选择根目录即可。

3

上传网站文件

点击"文件"菜单，进入网站目录，上传你的 HTML 文件，或者通过 1Panel 在线编辑。

💡 小白提示

如果你没有域名，可以直接用服务器 IP 访问。在创建网站时，主域名填写你的服务器 IP 即可。

需要数据库？1Panel 应用商店里有各种数据库，一键安装：

🔒 配置 HTTPS（SSL 证书）

让网站支持 HTTPS 非常简单，1Panel 支持自动申请免费证书：

1

进入网站设置

点击"网站"，找到你的网站，点击"配置"。

2

申请证书

切换到"HTTPS"选项卡，开启 HTTPS，选择"自动获取证书"，输入你的邮箱，保存即可。

3

完成

1Panel 会自动向 Let's Encrypt 申请证书并配置，通常只需几秒钟，证书到期会自动续期。

✅ 注意

申请证书需要域名已经解析到服务器 IP，且 80 端口可以访问。

💡 什么是 Docker？

简单说，Docker 就像是一个"软件盒子"，把应用和它需要的环境打包在一起，无论在哪台服务器上都能正常运行，不用担心环境配置问题。

❓ 常见问题

安装时提示 "curl: command not found" 怎么办？

这是因为你的系统没有安装 curl，先执行以下命令安装：

# CentOS/RHEL yum install -y curl # Ubuntu/Debian apt-get update && apt-get install -y curl
忘记 1Panel 密码怎么办？

在服务器上执行以下命令重置密码：

1pctl reset password

然后按提示输入新密码即可。

如何修改面板端口？

执行以下命令，按提示修改：

1pctl update port
网站无法访问怎么办？

按以下步骤排查：

  1. 检查 OpenResty 是否已启动（在"应用商店"→"已安装"中查看）
  2. 检查网站配置中的端口是否正确（默认 80/443）
  3. 检查服务器安全组/防火墙是否放行了对应端口
  4. 检查域名是否解析到服务器 IP
如何备份 1Panel 配置？

1Panel 支持快照功能，可以完整备份所有配置：

1. 点击左侧"快照"菜单

2. 点击"创建快照"

3. 选择备份到本地或其他存储（如阿里云 OSS）

需要恢复时，点击"恢复快照"即可。

如何卸载 1Panel？

如果需要卸载，执行以下命令：

# 停止 1Panel 1pctl stop # 卸载（会删除所有数据，请提前备份！） rm -rf /opt/1panel

🎓 进阶技巧

📅 设置自动备份

在"计划任务"中，可以设置：

  • 自动备份网站文件到云存储
  • 定期备份数据库
  • 自动清理过期备份（节省空间）
  • 定时执行 Shell 脚本

🔐 两步验证（2FA）

为了安全，建议开启两步验证：

1. 点击右上角头像 → "安全设置"

2. 开启"两步验证"

3. 用 Google Authenticator 或 Authy 扫描二维码

4. 以后登录除了密码，还需要输入手机上的验证码

🌐 绑定域名访问面板

直接用 IP:端口访问面板不够安全，建议绑定域名：

1. 创建一个网站，绑定你的域名（如 panel.yourdomain.com）

2. 在网站配置中设置"反向代理"，目标地址填 http://127.0.0.1:9999

3. 开启 HTTPS

4. 以后就可以用域名访问面板了！
