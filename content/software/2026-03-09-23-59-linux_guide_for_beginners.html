🤔什么是 Linux？一句话解释

💡 通俗理解：Linux 是一个免费、开源的操作系统，就像 Windows 和 macOS 一样，但它更强大、更安全、更适合程序员和技术人员使用。

打个比方。

  • 🪟 Windows 就像自动挡汽车 - 简单易用，但可控性有限
  • 🍎 macOS 就像豪华轿车 - 颜值高，但价格贵
  • 🐧 Linux 就像手动挡跑车 - 需要学习，但性能强劲，完全可控

📦Linux 发行版是什么？

Linux 有很多不同的「版本」，叫做发行版（Distribution）。就像 Android 有小米、华为、三星等不同品牌的手机系统一样。

🎯 小白推荐：新手建议选择 Ubuntu 或 Linux Mint，社区活跃，遇到问题容易找到解决方案。

🚀如何开始体验 Linux？（三种方式）

WSL - Windows Subsystem for Linux

Windows 10/11 内置的 Linux 子系统，不用重启电脑就能用 Linux。

PowerShell
# 1. 以管理员身份打开 PowerShell，输入以下命令安装 WSL
PS C:\> wsl --install

# 2. 重启电脑
# 3. 在开始菜单找到 Ubuntu，点击打开
# 4. 设置用户名和密码（输入密码时不显示，这是正常的）
✅ 优点：简单方便，不影响现有系统，可以随时切换 Windows 和 Linux
❌ 缺点：无法使用完整的图形界面（但命令行学习足够了）

虚拟机 - VirtualBox / VMware

在 Windows 里安装一个「虚拟电脑」，在里面运行 Linux。

  1. 下载 VirtualBox 并安装
  2. 下载 Ubuntu ISO 镜像从 ubuntu.com
  3. 在 VirtualBox 中新建虚拟机
  4. 分配 2GB 内存 + 20GB 硬盘空间
  5. 选择下载的 ISO 文件启动安装
⚠️ 注意：虚拟机需要电脑有较好的配置（建议 8GB 内存以上）

双系统 - 最完整的体验

电脑开机时可以选择进入 Windows 还是 Linux。

  1. 准备一个 8GB 以上的 U 盘
  2. 下载 Ubuntu ISO 镜像
  3. 使用 Rufus 制作启动盘
  4. 压缩磁盘分区，留出未分配空间
  5. 重启电脑，按 F12（或 F2/DEL）进入 BIOS
  6. 选择 U 盘启动，按提示安装
⚠️ 重要：安装双系统有风险，建议先备份重要数据，或找有经验的人帮忙。

🗂️Linux 文件系统结构

Linux 的文件系统和 Windows 很不一样，没有 C盘 D盘，所有东西都从根目录 / 开始。

📁/ (根目录)
├── 📁/bin - 基本命令（ls, cat等）
├── 📁/boot - 启动文件
├── 📁/dev - 设备文件
├── 📁/etc - 配置文件
├── 📁/home - 用户主目录
│ ├── 📁/home/user1 - 用户1的文件
│ └── 📁/home/user2 - 用户2的文件
├── 📁/lib - 系统库文件
├── 📁/mnt - 挂载点（U盘等）
├── 📁/opt - 可选软件
├── 📁/tmp - 临时文件
├── 📁/usr - 用户程序
└── 📁/var - 日志文件
💡 重要提示：
  • /home/用户名 相当于 Windows 的「我的文档」
  • ~ 是 /home/用户名 的简写
  • . 表示当前目录，.. 表示上级目录

🔐文件权限管理

Linux 是一个多用户系统，权限管理非常重要。每个文件都有对应的读、写、执行权限。

Terminal
user@ubuntu:~$ ls -l file.txt
-rw-r--r-- 1 user group 1234 Mar 9 10:00 file.txt
│└┬┘└┬┘└┬┘
│ │ │ └── 其他人的权限
│ │ └───── 所属组的权限
│ └──────── 文件所有者的权限
└────────── 文件类型（-表示普通文件，d表示目录）

权限字符含义

字符
含义
数字
r
读权限（Read）
4
w
写权限（Write）
2
x
执行权限（eXecute）
1
-
无该权限
0
🔢 权限数值计算：
rwx = 4+2+1 = 7
rw- = 4+2+0 = 6
r-x = 4+0+1 = 5
r-- = 4+0+0 = 4

修改权限

Terminal
# 给 file.sh 添加可执行权限
user@ubuntu:~$ chmod +x file.sh

# 设置权限为rwxr-xr-x（所有者可读写执行，其他人可读执行）
user@ubuntu:~$ chmod 755 file.sh

# 修改文件所有者
user@ubuntu:~$ sudo chown root:root file.sh

# 常用权限组合
user@ubuntu:~$ chmod 644 file.txt # rw-r--r--（普通文件）
user@ubuntu:~$ chmod 755 folder/ # rwxr-xr-x（可执行/目录）
user@ubuntu:~$ chmod 600 secret.key # rw-------（仅自己可读写）

📦软件包管理（安装软件）

Linux 使用包管理器来安装、更新和卸载软件，比 Windows 下载 exe 安装更方便、更安全。

# 更新软件包列表
$ sudo apt update

# 升级所有已安装的软件
$ sudo apt upgrade

# 安装软件
$ sudo apt install nginx

# 卸载软件
$ sudo apt remove nginx

# 搜索软件包
$ apt search python3

# 清理不需要的包
$ sudo apt autoremove
# 更新软件
$ sudo yum update

# 安装软件
$ sudo yum install nginx

# 卸载软件
$ sudo yum remove nginx

# 搜索软件包
$ yum search python3
# 更新系统
$ sudo pacman -Syu

# 安装软件
$ sudo pacman -S nginx

# 卸载软件
$ sudo pacman -R nginx

# 搜索软件包
$ pacman -Ss python3

❓常见问题 FAQ

Q1

我输入密码为什么不显示？

这是 Linux 的安全特性，输入密码时不会显示任何字符（包括星号），直接输入完按回车即可。

Q2

命令输错了怎么办？

按 Ctrl + C 可以中断当前命令，然后重新输入。

Q3

如何退出 Vim 编辑器？

先按 Esc 键，然后输入 :q 回车（不保存），或 :wq 回车（保存并退出）。

如果卡住，按 Esc 然后 :q! 强制退出。

Q4

sudo 和 su 有什么区别？

  • sudo：临时使用管理员权限执行单个命令
  • su：切换到 root 用户（需要 root 密码）

日常使用建议用 sudo，更安全。

Q5

如何查看命令的帮助文档？

  • man 命令名 - 查看详细手册（如 man ls）
  • 命令名 --help - 查看简要帮助（如 ls --help）
