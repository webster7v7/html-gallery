📖 什么是 Arch Linux？

Arch Linux 是一款轻量级、灵活的 Linux 发行版，遵循简洁、现代、实用、以用户为中心的设计理念。

为什么选择 Arch Linux？

  • 滚动更新 - 永远使用最新版本的软件，无需重装系统
  • 轻量简洁 - 只安装你需要的东西，没有臃肿的预装软件
  • 丰富的软件库 - 官方仓库 + AUR（Arch 用户仓库）拥有海量软件
  • 学习价值高 - 深入了解 Linux 系统的绝佳选择
  • 活跃的社区 - Arch Wiki 是最全面的 Linux 文档之一
注意：Arch Linux 需要一定的动手能力，安装过程使用命令行。但别担心，跟着本教程一步步来，你一定能成功！

💿 安装 Arch Linux

准备工作

  1. 下载 Arch Linux ISO 镜像：官网下载
  2. 制作启动 U 盘（推荐工具：Rufus、Ventoy 或 balenaEtcher）
  3. 备份重要数据！
  4. 确保电脑已连接网络（有线网络最简单）

详细安装步骤

1启动到安装环境

插入 U 盘，从 U 盘启动，选择 "Arch Linux install medium"。成功后会进入命令行界面。

如果键盘布局不是美式键盘，可以修改：loadkeys cn（中文键盘）

2检查网络连接

📟 终端
ping archlinux.org

如果显示数据包返回，说明网络正常。按 Ctrl+C 停止。

如果是 WiFi，使用：iwctl 进行连接

3更新系统时间

📟 终端
timedatectl set-ntp true

4分区（重要！）

查看磁盘：

📟 终端
fdisk -l

找到你的硬盘（通常是 /dev/sda 或 /dev/nvme0n1）

注意：分区会清除数据！请确认选择正确的磁盘！

使用 cfdisk 分区（适合新手）：

📟 终端
cfdisk /dev/sda

推荐分区方案（UEFI）：

分区 大小 类型 用途
/dev/sda1 512MB EFI System 启动分区
/dev/sda2 剩余空间 Linux filesystem 根分区

写入后选择 Quit 退出

5格式化分区

📟 终端
# 格式化 EFI 分区
mkfs.fat -F32 /dev/sda1

# 格式化根分区（ext4）
mkfs.ext4 /dev/sda2

6挂载分区

📟 终端
# 挂载根分区
mount /dev/sda2 /mnt

# 创建并挂载 EFI 分区
mkdir -p /mnt/boot/EFI
mount /dev/sda1 /mnt/boot/EFI

7安装基础系统

📟 终端
pacstrap /mnt base base-devel linux linux-firmware vim nano networkmanager

这会安装基础系统、Linux 内核、常用工具和编辑器。

8生成 fstab 文件

📟 终端
genfstab -U /mnt >> /mnt/etc/fstab

9进入新系统

📟 终端
arch-chroot /mnt

10设置时区

📟 终端
# 查看时区
ls /usr/share/zoneinfo/Asia

# 设置时区（中国）
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
hwclock --systohc

11设置语言

📟 终端
# 编辑 locale.gen
nano /etc/locale.gen

找到并取消注释（删除 #）：

en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8

保存（Ctrl+O，回车，Ctrl+X），然后执行：

📟 终端
locale-gen
echo "LANG=zh_CN.UTF-8" > /etc/locale.conf

12设置主机名

📟 终端
# 设置主机名（myarch 可以改成你喜欢的名字）
echo "myarch" > /etc/hostname

# 编辑 hosts 文件
nano /etc/hosts

添加以下内容：

127.0.0.1   localhost
::1         localhost
127.0.1.1   myarch.localdomain   myarch

13设置 root 密码

📟 终端
passwd

输入两次密码（不会显示）

14安装引导程序（GRUB）

📟 终端
# 安装 GRUB 和相关包
pacman -S grub efibootmgr

# 安装 GRUB
grub-install --target=x86_64-efi --efi-directory=/boot/EFI --bootloader-id=GRUB

# 生成配置文件
grub-mkconfig -o /boot/grub/grub.cfg

15启用网络服务

📟 终端
systemctl enable NetworkManager

16退出并重启

📟 终端
# 退出 chroot 环境
exit

# 卸载分区
umount -R /mnt

# 重启
reboot

记得拔出 U 盘！

🎉 恭喜！ Arch Linux 基础安装完成！重启后使用 root 账号和刚才设置的密码登录。

⚙️ 基础配置

1. 创建普通用户

日常使用不建议用 root，创建一个普通用户：

📟 终端
# 创建用户（yourname 换成你的名字）
useradd -m -G wheel yourname

# 设置密码
passwd yourname

# 安装 sudo
pacman -S sudo

# 赋予 sudo 权限
EDITOR=nano visudo

找到 # %wheel ALL=(ALL:ALL) ALL 这一行，取消注释（删除 #），保存退出。

2. 添加 Arch Linux CN 源（推荐）

Arch Linux 中文社区源包含很多常用中文软件：

📟 终端
# 编辑 pacman 配置
nano /etc/pacman.conf

在文件末尾添加：

[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

然后导入 GPG key：

📟 终端
pacman -Syy
pacman -S archlinuxcn-keyring

3. 更换国内镜像源（加速下载）

📟 终端
# 安装 reflector
pacman -S reflector

# 自动生成最快镜像源列表
reflector --country China --protocol https --latest 10 --sort rate --save /etc/pacman.d/mirrorlist

4. 安装微码（重要）

📟 终端
# Intel CPU
pacman -S intel-ucode

# AMD CPU
pacman -S amd-ucode

# 重新生成 GRUB 配置
grub-mkconfig -o /boot/grub/grub.cfg

📦 软件包管理

Arch Linux 使用 pacman 作为包管理器，以下是常用命令：

pacman 常用命令

命令 作用
pacman -S 包名 安装软件包
pacman -R 包名 删除软件包
pacman -Rs 包名 删除软件包及依赖
pacman -Syu 更新系统（同步仓库 + 升级）
pacman -Sy 同步软件仓库
pacman -Ss 关键词 搜索软件包
pacman -Qi 包名 查看已安装包信息
pacman -Ql 包名 查看包安装的文件
pacman -Sc 清理软件包缓存
更新系统的正确姿势：定期执行 sudo pacman -Syu 保持系统最新。Arch 是滚动更新，建议每周至少更新一次。

AUR（Arch 用户仓库）

AUR 包含大量用户贡献的软件包，使用 yay 工具可以方便地安装：

📟 终端
# 安装 yay（使用 archlinuxcn 源）
sudo pacman -S yay

# 或者从 AUR 安装
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

yay 常用命令：

命令 作用
yay -S 包名 从 AUR/官方仓库安装
yay -R 包名 删除软件包
yay -Syu 更新所有包（包括 AUR）
yay 关键词 搜索并交互式安装
yay -Yc 清理不需要的依赖

🖥️ 安装桌面环境

Arch 默认只有命令行，需要手动安装桌面环境。以下是几种热门选择：

方案一：KDE Plasma（推荐新手）

现代化、功能丰富、美观的桌面环境：

📟 终端
# 安装 KDE Plasma 和常用应用
sudo pacman -S plasma kde-applications sddm

# 启用显示管理器
sudo systemctl enable sddm

# 安装中文字体
sudo pacman -S noto-fonts-cjk wqy-microhei

方案二：GNOME

简洁、现代的桌面环境：

📟 终端
# 安装 GNOME
sudo pacman -S gnome gnome-tweaks gdm

# 启用显示管理器
sudo systemctl enable gdm

方案三：XFCE（轻量级）

适合老旧电脑：

📟 终端
# 安装 XFCE
sudo pacman -S xfce4 xfce4-goodies lightdm lightdm-gtk-greeter

# 启用显示管理器
sudo systemctl enable lightdm

方案四：i3wm（平铺窗口管理器）

适合键盘党、极客：

📟 终端
# 安装 i3
sudo pacman -S i3-wm i3status dmenu rxvt-unicode lightdm lightdm-gtk-greeter

# 启用显示管理器
sudo systemctl enable lightdm
安装完桌面环境后，重启即可进入图形界面：sudo reboot

💡 实用技巧

1. 系统维护

📟 终端
# 更新系统
sudo pacman -Syu

# 清理缓存（保留最近3个版本）
sudo paccache -r

# 删除孤立包
sudo pacman -Rns $(pacman -Qtdq)

# 查看系统信息
neofetch

2. 解决更新问题

📟 终端
# 如果更新失败，先同步密钥环
sudo pacman -Sy archlinux-keyring

# 强制刷新数据库
sudo pacman -Syyu

# 修复损坏的包
sudo pacman -S --overwrite="*" 包名
