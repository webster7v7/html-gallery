🎯 什么是 Live2D？

Live2D 是一种革命性的2D角色动画技术，它能够让静态的2D图像产生类似3D的立体动态效果，无需进行复杂的3D建模。

💡 核心原理

Live2D 通过对角色的2D原画进行分层切片，然后为每个部件添加变形控制点，最后通过参数驱动这些控制点来实现各种表情和动作。

🌟 Live2D 的应用场景

  • 虚拟主播 (VTuber) - 通过面捕技术实时驱动角色表情
  • 游戏角色 - 视觉小说、恋爱游戏、RPG中的动态立绘
  • App/网站看板娘 - 可交互的网页角色
  • 数字标牌/展示 - 商场、展会中的互动展示
  • 视频制作 - 省时省力的2D动画视频

🎀 学习 Live2D 需要什么基础？

好消息是：几乎零门槛！ 你不需要会画画，也不需要懂编程。只要有耐心学习软件操作，就能做出不错的作品。当然，如果有绘画基础会让你在角色设计方面更有优势。

🎨 前期准备：原画素材

📐 原画要求

Live2D 需要一张分层的 PSD 文件作为素材源。如果你不会画画，可以：

  • 使用官方提供的示例模型练习
  • 委托画师绘制（约稿）
  • 使用免费/付费的Live2D用立绘素材

⚠️ 重要：图层分组规范

原画的图层分组直接影响建模效率。建议在建模前就和画师沟通好图层结构，或者自己整理图层。

📋 标准图层结构

📁 角色名称/ ├── 📁 头部/ │ ├── 📁 前发/ （刘海，最外层） │ ├── 📁 脸部/ │ │ ├── 脸轮廓 │ │ ├── 眉毛_左 │ │ ├── 眉毛_右 │ │ ├── 眼睛_左/ │ │ │ ├── 眼白 │ │ │ ├── 眼珠 │ │ │ └── 高光 │ │ ├── 眼睛_右/ │ │ ├── 鼻子 │ │ └── 嘴巴/ │ │ ├── 口腔 │ │ ├── 牙齿 │ │ └── 舌头 │ ├── 📁 后发/ （后脑勺头发） │ └── 📁 配件/ （发饰、眼镜等） └── 📁 身体/ ├── 脖子 ├── 身体 ├── 📁 手臂_左/ └── 📁 手臂_右/

✅ 原画绘制要点

🎀 头部绘制规范

  • 五官位置：眼睛在头部中线偏下，鼻子在眼睛和下巴中间
  • 图层遮挡：头发要画全，包括被脸挡住的部分（用于转头时显示）
  • 闭眼状态：建议额外绘制闭眼图层，或闭眼时眼睛不会变形的结构
  • 嘴巴内部：画闭嘴时也要画出口腔内部，方便做张嘴动画

💡 身体绘制规范

  • 关节处：手臂、腿的关节处要画得圆润，方便变形
  • 身体遮挡：被头发遮挡的身体部分也要画完整
  • 对称部件：左右对称的部件可以先画一边，另一边复制翻转

🎯 画布尺寸建议

  • 一般用途：2000×2000 像素以上
  • VTuber使用：建议 3000×3000 像素以上，确保高清
  • 分辨率：72~150 DPI 即可，屏幕显示不需要印刷级DPI

🎭 变形器 (Deformer) 详解

变形器是 Live2D 的核心概念，它负责控制部件的变形。主要有两种类型：

🔷 曲面变形器 (Warp Deformer)

使用网格控制变形，适合做曲线变形（如头发飘动、身体弯曲）。是最常用的变形器类型。

🔲 旋转变形器 (Rotation Deformer)

以某一点为中心进行旋转，适合做关节运动（如手臂摆动、头部转动）。

📐 曲面变形器创建步骤

选中部件

在部件面板中选中要添加变形器的图层（如刘海）。

创建变形器

点击菜单「编辑」→「创建曲面变形器」，或按 Ctrl+Shift+C。

设置网格

在弹窗中设置网格数量。新手建议 3×3 或 4×4，后期可调整。

调整控制点

拖动控制点调整部件形状，观察变形效果。

🔄 旋转变形器创建步骤

选中部件

选择需要做旋转运动的部件（如头部、手臂）。

创建变形器

点击菜单「编辑」→「创建旋转变形器」，或按 Ctrl+Shift+R。

设置旋转中心

点击确定旋转中心点（如头部的脖子位置）。

测试旋转

拖动控制点测试旋转效果，调整范围和限制。

🎀 变形器层级技巧

变形器可以嵌套！例如：

  • 头部整体 → 旋转变形器（控制转头）
  • 　└ 头发 → 曲面变形器（控制头发飘动）
  • 　　　└ 发梢 → 另一个曲面变形器（更细腻的摆动）

合理的层级结构能让动画更自然！

✏️ 编辑变形器的技巧

  • 添加控制点：选中变形器，按 Ctrl 点击边缘可添加控制点
  • 删除控制点：选中点后按 Delete
  • 调整网格密度：右键变形器 →「编辑变形器」→ 修改分割数
  • 多个点同时移动：用套索工具框选多个点后统一拖动

🎚️ 参数系统 (Parameter)

参数是控制 Live2D 动画的"开关"和"滑块"。每个参数对应一种动作或表情，通过改变参数值来驱动变形器。

💡 参数的理解方式

想象参数就是一个从 -1 到 1（或 0 到 1）的滑块：

  • 角度 X：-1（左转）←→ 0（正面）←→ 1（右转）
  • 眼睛开闭：0（闭眼）←→ 1（睁眼）
  • 嘴巴开合：0（闭嘴）←→ 1（张嘴）

📋 标准参数列表（VTuber常用）

参数名 范围 说明
Angle X / 角度 X -30 ~ 30 头部左右转动
Angle Y / 角度 Y -30 ~ 30 头部上下点头
Angle Z / 角度 Z -30 ~ 30 头部左右倾斜
Eye L Open 0 ~ 1 左眼开闭
Eye R Open 0 ~ 1 右眼开闭
Eye L Smile 0 ~ 1 左眼笑眼程度
Eye R Smile 0 ~ 1 右眼笑眼程度
Mouth Open 0 ~ 1 嘴巴开合
Mouth Form -1 ~ 1 嘴型变化（-1笑 ~ 1生气）
Brow L Y / R Y -1 ~ 1 眉毛上下
Brow L X / R X -1 ~ 1 眉毛左右移动
Brow L Angle / R Angle -1 ~ 1 眉毛角度
Body X / Y -10 ~ 10 身体随头部移动
Breath 0 ~ 1 呼吸动画

📝 创建参数的三种方式

方式一：使用标准参数（推荐新手）

  1. 打开「参数」面板
  2. 点击右上角菜单 →「追加标准参数」
  3. 勾选需要的参数类型（如脸部、身体等）
  4. 自动创建一整套标准参数

方式二：从模板导入

  1. 菜单「文件」→「打开」
  2. 选择官方示例模型或模板
  3. 复制需要的参数到自己的项目

方式三：手动创建

  1. 参数面板点击「+」按钮
  2. 输入参数名称（如"Hair Wave"）
  3. 设置最小值、最大值、默认值
  4. 绑定到对应的变形器关键帧

🔗 将参数绑定到变形器

进入编辑模式

选中要绑定的变形器，确保参数面板可见。

设置关键点

点击参数左侧的「●」圆点，使其变成「◉」（选中状态）。

调整变形

拖动变形器的控制点，调整到该参数值对应的状态。

设置另一端

将参数滑块拖到另一端（如从-1到1），调整对应的变形状态。

✅ 绑定检查

绑定完成后，来回拖动参数滑块，检查变形是否流畅自然。如果变形不理想，可以随时调整控制点位置。

🎬 动画制作

🎞️ 时间轴基础

Live2D 使用关键帧动画系统。你在不同时间点设置参数值（关键帧），软件会自动计算中间的过渡。

📝 制作眨眼动画（入门练习）

打开动画工作区

点击顶部「动画」标签切换到动画模式。

新建动画文件

「文件」→「新建」→「动画」，设置帧率（通常30fps）。

放置模型

从部件面板拖拽模型到时间轴上。

设置睁眼关键帧

在第1帧，确保「Eye Open」参数为1，点击参数右侧的「●」记录关键帧。

设置闭眼关键帧

在第15帧，将「Eye Open」改为0，再次记录关键帧。

设置恢复关键帧

在第30帧，将「Eye Open」改回1，完成一次眨眼循环。

💡 循环播放

在时间轴上设置循环范围（拖动时间轴上方的标记），点击播放按钮预览循环动画。

🎭 物理效果（自动动画）

Live2D 内置物理引擎，可以让头发、耳朵、尾巴等部件自动跟随运动，无需手动设置每一帧！

设置步骤：

  1. 打开「物理/场景混合设定」面板（菜单「窗口」→「物理/场景混合设定」）
  2. 点击「追加」创建新的物理设定
  3. 输入端：选择驱动源（如「Angle X」头部转动）
  4. 摆锤设定：设置摆锤数量、长度、重量等参数
  5. 输出端：选择要驱动的变形器（如头发的曲面变形器）
  6. 预览并调整参数直到满意

🎀 物理参数调整建议

  • 输入影响度：头部转动对头发的影响程度
  • 摆锤长度：越长摆动幅度越大
  • 重力方向：通常设置为向下（Y轴负方向）
  • 影响力：控制物理效果的强度

🎤 面部表情系统（Expression）

Live2D 支持将常用表情保存为「表情预设」，方便在动画中快速调用。

创建表情预设：

  1. 切换到「建模」模式
  2. 调整参数到你想要的表情（如微笑：嘴角上扬、眼睛微眯）
  3. 在「表情」面板点击「+」
  4. 输入表情名称（如「微笑」「惊讶」「生气」）
  5. 点击「设定」保存当前参数状态

📤 导出与应用

💾 导出格式选择

格式 用途 说明
.cmo3 工程文件 可再次编辑的源文件，保存所有图层和参数
.moc3 运行时模型 最终使用的模型文件，体积小、加载快

🎯 导出 .moc3 模型（用于直播/应用）

检查模型

确保所有参数绑定正确，物理效果设置完成。

执行导出

菜单「文件」→「导出运行档」→「导出 moc3 文件」。

选择文件夹

选择一个空文件夹，所有相关文件会一起导出。

检查导出内容

确认文件夹中包含：.moc3、.model3.json、贴图文件(.png)等。

📹 导出视频（用于视频制作）

  1. 在动画模式下，完成你的动画制作
  2. 「文件」→「导出视频」→ 选择格式（MP4/GIF/序列帧）
  3. 设置分辨率、帧率（建议1920×1080，30fps）
  4. 选择导出范围（全部/指定范围）
  5. 点击导出等待渲染完成

💡 实用技巧与避坑指南

🚀 提升效率的技巧

⚡ 快捷键汇总

  • Ctrl+Z / Ctrl+Y - 撤销/重做
  • 1 / 2 / 3 - 切换编辑模式（建模/动画/纹理）
  • F - 选中部件居中显示
  • Tab - 显示/隐藏面板
  • Space+拖拽 - 平移画布
  • Ctrl++ / Ctrl+- - 缩放画布
  • Ctrl+Shift+C - 创建曲面变形器
  • Ctrl+Shift+R - 创建旋转变形器

✅ 建模最佳实践

  • 从大到小：先做整体（头、身体），再做细节（五官、手指）
  • 对称操作：先完成左边，复制翻转做右边，保持一致性
  • 经常预览：多拖动参数测试，及时发现问题
  • 分步保存：重要节点另存版本，防止误操作
  • 参考模板：学习官方示例的分层和参数设置

⚠️ 常见问题与解决方案

问题：部件边缘出现缝隙

原因：原画图层之间有空隙

解决：在绘图软件中确保相邻图层有重叠区域，或者使用「蒙版」功能

问题：变形后出现锯齿/扭曲

原因：网格密度不够或控制点分布不合理

解决：增加曲面变形器的网格分割数，或者重新布置控制点

问题：参数绑定后没有反应

原因：关键帧没有正确设置

解决：检查是否点击了参数左侧的「●」进入编辑模式，确保两端都有设置关键点

问题：导出后模型显示异常

原因：贴图路径错误或文件缺失

解决：确保导出的文件夹包含所有资源，检查 .model3.json 中的路径配置
