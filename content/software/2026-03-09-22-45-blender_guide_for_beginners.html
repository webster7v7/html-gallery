🤔 什么是 Blender？

Blender 是一款免费开源的 3D 创作软件，功能涵盖建模、雕刻、动画、渲染、合成、视频剪辑等全流程。无论你是想做 3D 模型、游戏资产、动画短片还是视觉特效，Blender 都能胜任。

💾 下载与安装

🚀 最新版本：Blender 4.x

支持 Windows、macOS、Linux，无需注册直接下载

前往官网下载

💡 建议下载稳定版本（Stable），测试版（Alpha/Beta）可能有 Bug

安装步骤

  1. 下载安装包

    根据你的系统选择对应版本下载

    • Windows: 下载 .msi 安装程序
    • macOS: 下载 .dmg 镜像（支持 Intel 和 Apple Silicon）
    • Linux: 下载 .tar.xz 压缩包或使用包管理器
  2. 运行安装

    Windows 直接运行安装程序；macOS 拖入应用程序文件夹；Linux 解压后运行 blender

  3. 首次启动

    打开 Blender，选择语言为简体中文，建议启用"自动保存"功能

💡 配置建议：Blender 对显卡有一定要求，建议至少有 8GB 内存，独立显卡（NVIDIA/AMD）可加速渲染。如果电脑配置较低，可以在首选项中调低视窗质量。

🖥️ 界面介绍

Blender 默认界面采用分区布局，每个区域都可以自由调整。首次打开可能会觉得复杂，但熟悉后会发现非常高效。

📍 主要区域说明

3D 视窗（中间最大区域）

场景的主要操作区域，可以旋转、缩放、平移视角查看模型

工具栏（左侧）

选择、移动、旋转、缩放等基本工具

属性面板（右侧）

物体的属性、材质、修改器、渲染设置等

时间轴（底部）

动画制作时使用，显示帧数和时间

大纲视图（右上角）

场景中的物体列表，可以管理物体的层级关系

界面操作基础

操作 鼠标操作 说明
旋转视角 中键 拖拽 围绕场景中心旋转查看
平移视角 Shift + 中键 拖拽 上下左右移动视角
缩放视角 滚轮 滚动 放大或缩小视图
框选放大 Shift + B + 框选 框选区域放大查看
视角复位 小键盘 0/1/3/7 切换到相机/正交视图
🎯 重要提示：Blender 默认使用"中键"（鼠标滚轮按下）进行视角旋转。如果你的鼠标没有中键，可以在 编辑 → 偏好设置 → 输入 → 鼠标 中开启"模拟 3 键鼠标"，然后用 Alt + 左键 代替。

🎮 基础操作

物体基本变换

选中物体后（左键 点击），可以使用以下快捷键：

快捷键 功能 操作说明
G 移动 (Grab) Grab - 抓取移动，移动鼠标后 左键 确认
R 旋转 (Rotate) 旋转物体，可配合 X/Y/Z 锁定轴向
S 缩放 (Scale) 缩放物体，左键 确认，右键 取消
X/Y/Z 锁定轴向 变换时按可锁定对应轴向
Shift + D 复制 Duplicate - 复制选中物体
Tab 编辑/物体模式切换 编辑模式可以编辑顶点、边、面
💡 操作技巧：按 G/R/S 后，可以直接按 X、Y 或 Z 锁定对应轴向。再按一次数字可以精确输入数值，如 G → Z → 2 → Enter 表示沿 Z 轴向上移动 2 个单位。

常用编辑操作

快捷键 功能
A 全选 / 取消全选
Ctrl + I 反选
H 隐藏选中
Alt + H 显示全部
Ctrl + Z 撤销
Ctrl + Shift + Z 重做
F12 渲染图像

🏗️ 建模入门

添加基础物体

按 Shift + A 打开添加菜单，可以选择添加各种基础几何体：

🧊

立方体 (Cube)

最基础的形状，适合建筑、箱子等

⚪

球体 (UV Sphere)

球状物体，适合行星、球类等

🛢️

圆柱 (Cylinder)

柱状物体，适合柱子、瓶子等

🍩

圆环 (Torus)

甜甜圈形状，适合轮胎、装饰等

编辑模式详解

按 Tab 进入编辑模式后，可以选择不同的选择模式：

顶点 (Vertex)
边 (Edge)
面 (Face)
快捷键 功能 说明
1 顶点选择模式 选择并移动单个点
2 边选择模式 选择并编辑边线
3 面选择模式 选择并编辑面片
E 挤出 (Extrude) 将选中的面/边拉伸出新的几何体
Ctrl + R 环切 在模型上增加边线细分
S + 0 合并到中心 将选中的顶点合并
M 合并菜单 选择合并方式（按距离、中心等）

🎯 实战：制作一个简单的小桌子

  1. 创建桌面

    按 Shift + A → 网格 → 立方体，然后按 S → Y → 2 → Enter 将桌面拉长

  2. 创建桌腿

    再添加一个立方体，按 S 缩放成细长的形状，然后按 G 移动到角落

  3. 复制桌腿

    选中桌腿，按 Shift + D 复制，G 移动到其他三个角落

  4. 调整高度

    选中所有桌腿，按 G → Z 向下移动到合适位置

  5. 命名整理

    在大纲视图中双击物体名称，改名为 "桌面"、"桌腿" 等

🎉 恭喜！ 你已经完成了第一个 3D 模型！按 F12 可以渲染看看效果。

🎨 材质与渲染

添加材质

让模型有颜色和质感，需要在属性面板中添加材质：

  1. 选中物体

    在 3D 视窗中 左键 点击选中要添加材质的物体

  2. 打开材质属性

    在右侧属性面板点击红色圆球图标（材质属性）

  3. 新建材质

    点击"新建"按钮创建新材质

  4. 设置颜色

    在"基础色"中选择你想要的颜色

  5. 调整参数

    可以调整粗糙度、金属度等参数改变质感

📊 常用材质参数
  • 基础色 (Base Color) - 物体的主要颜色
  • 粗糙度 (Roughness) - 0 是镜面，1 是磨砂
  • 金属度 (Metallic) - 0 是非金属，1 是纯金属
  • 透明度 (Alpha) - 设置透明效果

EEVEE vs Cycles 渲染器

⚡

EEVEE（实时）

渲染速度快，适合预览、游戏风格渲染

🎬

Cycles（光线追踪）

真实感强，适合高质量静帧、电影级渲染

切换渲染器：在右侧属性面板点击相机图标（渲染属性），在"渲染引擎"中选择。

快速渲染设置

操作 说明
F12 渲染当前相机视角的图像
Ctrl + F12 渲染动画序列
F11 查看上次渲染结果
Esc 取消渲染
💡 渲染小技巧：
  • 按 0（小键盘）切换到相机视角
  • 按 N 打开侧边栏，在"视图"中可以锁定相机视角
  • 在渲染属性中可以设置输出分辨率和采样数

💡 实用技巧与资源

🚀 效率提升技巧

🎯 快速对齐视图

~（波浪键）打开饼菜单，可以快速切换前后左右上下视图

🔍 快速聚焦

选中物体后按 .（小键盘小数点）将视图聚焦到选中物体

📐 精确数值输入

进行变换操作时，可以直接输入数字，如 S → 2 表示放大 2 倍

🎨 快速切换着色方式

Z 打开着色饼菜单，可快速切换线框、实体、材质预览、渲染模式

⚠️ 新手常见问题

Q: 物体不见了怎么办？

按 .（小键盘小数点）聚焦选中物体，或按 Home 键显示全部物体

Q: 无法选中物体？

检查是否在正确的模式下（物体模式），以及是否启用了"仅选择可见"

Q: 渲染出来是黑色的？

检查是否有灯光、相机视角是否正确、渲染设置中的曝光是否正常

Q: 软件卡顿？

可以降低视窗细分级别、关闭阴影、减少多边形数量，或升级硬件配置

🎯 学习路径建议

  1. 基础操作（1-2 周）

    熟悉界面、基本变换、视图操作

  2. 简单建模（2-3 周）

    学习编辑模式、挤出、环切等基础建模工具

  3. 材质与渲染（1-2 周）

    了解材质节点、灯光设置、基础渲染

  4. 深入学习（持续）

    雕刻、动画、几何节点、合成等专业领域

🌟 给新手的建议：不要试图一次学会所有功能，Blender 功能非常强大，建议先掌握基础建模，然后根据自己的兴趣方向（动画、渲染、雕刻等）深入学习。坚持练习，每天做一点，进步会很明显！
