❓ 什么是 Shell？

Shell 是一个命令行解释器，它是你和操作系统之间的翻译官。你在终端输入的命令，由 Shell 解释后告诉电脑执行什么操作。

Terminal
$ echo "Hello Shell"
Hello Shell
$ _

💡 为什么要学 Shell？

  • 效率提升：批量操作文件，一键完成重复任务
  • 远程管理：服务器只能通过命令行操作
  • 开发必备：Git、Docker、编译部署都需要
  • 显得专业：程序员的基本功

常见 Shell 类型

Shell 名称 特点 默认系统
Bash 最流行，功能全面，文档丰富 Linux、旧版 macOS
Zsh 功能强大，插件丰富，推荐 新版 macOS
Fish 友好易用，自动补全强大 需手动安装
PowerShell 微软开发，跨平台 Windows

🖥️ 如何打开终端

Windows

  • 方法一：搜索打开

    按 Win 键，输入 "cmd" 或 "PowerShell"，回车打开

  • 方法二：快捷键（推荐）

    Win + R，输入 "wt"（Windows Terminal）或 "cmd"，回车

  • 方法三：右键菜单

    在文件夹空白处按住 Shift + 右键，选择"在此处打开 PowerShell 窗口"

macOS

  • 方法一：Spotlight

    Cmd + 空格，输入 "Terminal"，回车

  • 方法二：Launchpad

    在"其他"文件夹中找到"终端"

Linux

  • 快捷键

    Ctrl + Alt + T 是大多数发行版的默认快捷键

💡 推荐终端工具

  • Windows：Windows Terminal（微软商店免费下载）
  • macOS：iTerm2（功能强大，免费）
  • 跨平台：Tabby、Warp（现代化界面）

⌨️ 基础命令

学习 Shell 从这几个命令开始，它们是你使用最频繁的：

ls

列出当前目录的文件和文件夹

ls -la # 显示所有文件（含隐藏）
pwd

显示当前所在路径（我在哪）

pwd # /home/username/Documents
cd

切换目录（去哪里）

cd Documents # 进入 Documents 文件夹
clear

清屏（快捷键 Ctrl + L）

clear # 屏幕变干净
man

查看命令手册（帮助文档）

man ls # 查看 ls 命令的详细说明
history

查看历史命令记录

history | tail -20 # 最近20条

命令格式说明

命令格式
命令 [选项] [参数]
# 示例：
ls -la /home
命令：ls（列出目录内容）
选项：-la（显示所有，长格式）
参数：/home（要查看的目录）

🧭 目录导航

理解文件系统结构，像在自己的城市认路一样：

特殊目录符号

.

当前目录

cd . # 原地不动
..

上级目录

cd .. # 返回上一级
~

用户主目录（家目录）

cd ~ # 回到家
/

根目录（系统最顶层）

cd / # 去根目录
-

上次所在的目录

cd - # 快速来回切换

常用导航示例

导航演示
$ pwd
/home/alice
$ cd Documents
$ pwd
/home/alice/Documents
$ cd ../.. # 上两级
$ pwd
/home
$ cd ~ # 回家
$ cd ~/Desktop # 直接去桌面

📁 文件与目录操作

创建与删除

mkdir 文件夹名

创建新目录

mkdir projects # 创建 projects 文件夹
touch 文件名

创建空文件

touch hello.txt # 创建空文本文件
rm 文件名

删除文件

rm hello.txt # 删除文件
rm -r 目录名

删除目录及其内容

rm -r projects # 删除整个文件夹

复制与移动

cp 源文件 目标

复制文件

cp file.txt backup.txt
cp -r 源目录 目标

复制目录

cp -r folder1 folder2
mv 源文件 目标

移动或重命名

mv old.txt new.txt # 重命名

查看文件内容

cat 文件名

显示全部内容（适合小文件）

cat readme.txt
less 文件名

分页查看（可上下翻页）

less large.log # 按 q 退出
head -n 10 文件

查看前10行

head -n 20 data.csv
tail -n 10 文件

查看后10行

tail -f app.log # 实时追踪

⚠️ 删除文件需谨慎

  • Shell 删除的文件不会进回收站，直接永久删除
  • rm -rf / 会删除整个系统，永远不要执行！
  • 建议先用 ls 确认要删除的内容，再执行删除

📝 文本处理三剑客

grep、sed、awk 是 Shell 文本处理的三大神器：

🔍 grep - 文本搜索

grep 示例
# 在文件中搜索包含 "error" 的行
$ grep "error" log.txt
# 忽略大小写搜索
$ grep -i "error" log.txt
# 显示行号
$ grep -n "error" log.txt
# 递归搜索整个目录
$ grep -r "TODO" src/
# 反向匹配（不含 error 的行）
$ grep -v "error" log.txt

✏️ sed - 流编辑器

sed 示例
# 替换文本（s/原内容/新内容/）
$ sed 's/foo/bar/' file.txt # 每行第一个 foo 换成 bar
# 全局替换（所有匹配）
$ sed 's/foo/bar/g' file.txt
# 删除第5行
$ sed '5d' file.txt
# 直接修改文件（-i）
$ sed -i 's/foo/bar/g' file.txt

🔧 awk - 文本分析

awk 示例
# 打印第一列（默认空格分隔）
$ awk '{print $1}' data.txt
# 打印多列
$ awk '{print $1, $3}' data.txt
# 指定分隔符（如逗号）
$ awk -F',' '{print $2}' data.csv
# 条件筛选
$ awk '$3 > 100 {print $1}' data.txt
# 计算行数
$ awk 'END {print NR}' file.txt

🔗 管道与重定向

管道和重定向是 Shell 最强大的特性，让命令之间可以协同工作：

管道 | （把左边输出传给右边）

管道示例
# 查看进程，筛选出 chrome
$ ps aux | grep chrome
# 查看文件，统计行数
$ cat file.txt | wc -l
# 多级管道
$ cat log.txt | grep error | wc -l
# 查看最近使用的命令，去重排序
$ history | cut -c 8- | sort | uniq

重定向

> 文件

输出到文件（覆盖）

echo "hello" > file.txt
>> 文件

输出到文件（追加）

echo "world" >> file.txt
< 文件

从文件输入

sort < unsorted.txt
2> 文件

错误输出到文件

cmd 2> error.log
&> 文件

标准输出和错误都重定向

cmd &> all.log
/dev/null

黑洞，丢弃输出

cmd > /dev/null 2>&1

💡 实际应用场景

  • 备份配置：crontab -l > crontab.backup
  • 记录日志：./script.sh > output.log 2>&1
  • 查找大文件：du -h | sort -rh | head -10
  • 统计代码行数：find . -name "*.js" | xargs wc -l

🔐 权限管理

Linux/macOS 每个文件都有权限控制，谁可以读、写、执行：

查看权限

权限查看
$ ls -l
-rw-r--r-- 1 user staff 1234 Jan 15 10:00 file.txt
drwxr-xr-x 2 user staff 64 Jan 15 09:50 folder
文件类型：- 普通文件，d 目录
权限位：rwx（读、写、执行）
三组权限：所有者、所属组、其他人

修改权限 chmod

chmod +x script.sh

添加可执行权限

让脚本可以运行
chmod 755 file

数字模式设置权限

rwxr-xr-x（所有者可读写执行）
chmod 644 file

普通文件权限

rw-r--r--（所有人可读，仅所有者可写）

权限数字对照

权限 数字 说明
r-- 4 只读
-w- 2 只写
--x 1 只执行
rw- 6 读写
rwx 7 读写执行
r-x 5 读和执行

修改所有者 chown

chown 示例
# 修改文件所有者（需要 sudo）
$ sudo chown username file.txt
# 同时修改所有者和组
$ sudo chown user:group file.txt
# 递归修改目录
$ sudo chown -R user:group folder/

⌨️ 高效快捷键

掌握这些快捷键，命令行效率提升 10 倍：

光标移动

快捷键 功能
Ctrl + A 移到行首
Ctrl + E 移到行尾
Ctrl + F 前移一个字符（同 →）
Ctrl + B 后移一个字符（同 ←）
Alt + F 前移一个单词
Alt + B 后移一个单词

编辑操作

快捷键 功能
Ctrl + D 删除光标处字符
Ctrl + H 删除前一个字符（同 Backspace）
Ctrl + W 删除前一个单词
Ctrl + U 删除光标前整行
Ctrl + K 删除光标后整行
Ctrl + Y 粘贴刚才删除的内容

历史命令

快捷键 功能
↑ / ↓ 浏览历史命令
Ctrl + R 搜索历史命令（再按继续搜）
Ctrl + G 取消搜索
!! 执行上一条命令
!n 执行第 n 条历史命令
!$ 上条命令的最后一个参数

其他常用

快捷键 功能
Tab 自动补全命令/路径
Ctrl + L 清屏（同 clear）
Ctrl + C 终止当前命令
Ctrl + Z 暂停当前命令（后台运行用 fg）
Ctrl + D 退出终端（光标在行首时）

💡 快捷键记忆技巧

  • Ctrl + A/E：A 是字母表第一个（开头），E 是 End（结尾）
  • Ctrl + F/B：F = Forward，B = Backward
  • Ctrl + U/K：U = Unix 风格删前面，K = KDE 风格删后面
