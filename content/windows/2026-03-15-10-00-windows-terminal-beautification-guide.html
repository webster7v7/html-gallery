一键美化流程

1

安装 Nerd Fonts（关键步骤）

不装字体就会出现方块乱码！推荐使用 CaskaydiaCove Nerd Font， 这是微软 Cascadia Code 的 Nerd 补丁版，与 Windows Terminal 完美匹配 [^7^]。

快速安装命令（推荐）

oh-my-posh font install CaskaydiaCove

* 需先安装 Oh My Posh，或从 nerdfonts.com 手动下载

❌ 安装前
➜ ~/project [main] □ □ □
✅ 安装后
➜ ~/project  main ✓
2

初始化配置文件

创建 PowerShell 配置文件并初始化 Oh My Posh。如果提示执行策略错误，需先运行 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser [^8^]。

# 创建配置目录
mkdir ~/Documents/PowerShell -Force
# 生成默认配置（使用 paradox 主题）
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH/paradox.omp.json" | Out-File ~/Documents/PowerShell/Microsoft.PowerShell_profile.ps1 -Encoding utf8
3

设置终端字体

在 Windows Terminal 设置中选择刚才安装的 Nerd Font，开启亚克力效果和透明度 [^2^]。

1
打开 Windows Terminal → 设置 (Ctrl+,)
2
配置文件 → PowerShell → 外观
3
字体选择：CaskaydiaCove Nerd Font
4
透明度：70% + 开启亚克力效果 (Acrylic)

进阶增强

📁

Terminal-Icons

为文件和文件夹添加图标，让 ls 命令显示文件类型图标 [^5^]。

Install-Module -Name Terminal-Icons -Repository PSGallery -Scope CurrentUser
# 然后在 $PROFILE 中添加：
Import-Module Terminal-Icons
🌿

Posh-Git

在提示符中显示详细的 Git 状态信息，包括分支、未提交更改等 [^8^]。

Install-Module posh-git -Scope CurrentUser
# 添加到 $PROFILE：
Import-Module posh-git
⚡

Zoxide + Fzf

智能目录跳转工具，记住你访问过的文件夹，支持模糊搜索快速跳转。

winget install zoxide
# 添加到 $PROFILE：
Invoke-Expression (& { (zoxide init powershell | Out-String) })

常见问题

❌ 无法加载配置文件，禁止执行脚本

PowerShell 默认禁止执行第三方脚本，需要修改执行策略。

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

⚠️ 图标显示为方块或乱码

Windows Terminal 字体未设置为 Nerd Font，或字体名称拼写错误 [^16^]。

正确名称示例：CaskaydiaCove NF / MesloLGM Nerd Font / JetBrainsMono Nerd Font

🔄 主题切换不生效

修改配置文件后需要重新加载，或检查主题文件路径是否正确。

. $PROFILE # 重新加载配置
Get-PoshThemes # 查看所有可用主题
