快速开始

三步完成基础美化，全程不超过 5 分钟

1

安装 Oh My Posh

使用 winget 一键安装主题引擎（推荐），或手动下载安装

winget install JanDeDobbeleer.OhMyPosh
安装后重启终端以加载环境变量
2

安装 Nerd Font 字体

解决图标显示为方框或乱码问题，推荐 MesloLGM 或 JetBrainsMono

oh-my-posh font install Meslo
手动安装备选： 访问 nerdfonts.com 下载 FiraCode 或 Cascadia Code，解压后右键安装所有 .ttf 文件
3

配置 PowerShell 配置文件

创建或编辑配置文件，让 Oh My Posh 每次自动加载

# 创建配置文件（如果不存在）
if (!(Test-Path -Path $PROFILE)) { New-Item -ItemType File -Path $PROFILE -Force }

# 用记事本编辑
notepad $PROFILE

在打开的记事本中添加以下内容：

oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\jandedobbeleer.omp.json" | Invoke-Expression

完整配置文件示例 ($PROFILE) 

# Oh My Posh 主题初始化
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\powerlevel10k_rainbow.omp.json" | Invoke-Expression

# 导入模块
Import-Module Terminal-Icons
Import-Module posh-git
Import-Module PSReadLine

# PSReadLine 配置
Set-PSReadLineOption -PredictionSource History
Set-PSReadLineOption -PredictionViewStyle ListView
Set-PSReadLineKeyHandler -Key Tab -Function MenuComplete
Set-PSReadLineKeyHandler -Key UpArrow -Function HistorySearchBackward
Set-PSReadLineKeyHandler -Key DownArrow -Function HistorySearchForward

# 别名设置
Set-Alias -Name ll -Value Get-ChildItem
Set-Alias -Name g -Value git

# 欢迎语
Write-Host "🚀 PowerShell 已加载完成！" -ForegroundColor Green

常见问题

遇到问题？这里有解决方案

图标显示为方框或乱码？

原因：未安装 Nerd Font 或终端字体设置不正确。

  • 确认已安装 MesloLGM Nerd Font 等字体
  • Windows Terminal 设置中必须选择该字体
  • VS Code 需单独设置 terminal.integrated.fontFamily

提示 "无法加载配置文件"？

执行策略限制导致无法运行脚本。

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

终端启动变慢？

Oh My Posh 可能因网络或配置导致延迟。

  • 使用本地主题文件而非远程 URL
  • 禁用不必要的模块导入
  • 更新到最新版本：winget upgrade JanDeDobbeleer.OhMyPosh

CMD 也能用 Oh My Posh 吗？

可以，需要安装 Clink 工具。

winget install clink
# 在 clink 目录创建 oh-my-posh.lua
load(io.popen('oh-my-posh init cmd'):read("*a"))()
