什么是 winget？

winget（Windows Package Manager）是微软官方推出的命令行包管理工具。就像手机上的应用商店一样，但用命令行操作——输入一条命令，自动搜索、下载、安装软件，无需手动找官网、点下一步。

Windows 10/11 自带，无需额外安装
官方软件源，安全无毒
支持数千款常用软件
传统方式 vs winget 效率提升 10x

打开浏览器 → 搜索软件官网 → 找下载按钮 → 下载.exe → 双击运行 → 一路下一步 → 完成

winget install vscode
# 回车，30秒搞定

第一步：检查是否已安装

Windows 10 (1809+) 和 Windows 11 已内置 winget。按 Win + R，输入 cmd 回车，然后输入：

winget --version

如果显示版本号（如 v1.9.25200），说明已就绪！

如果提示"不是内部或外部命令"，去 Microsoft Store 搜索 "App Installer" 并更新。

核心命令 · 5个够用了

搜索软件

winget search chrome

找软件，看有哪些版本可用

安装软件

winget install vscode

自动下载并安装，静默安装加 --silent

查看已安装

winget list

列出系统上所有已安装的软件

升级软件

winget upgrade --all

升级所有软件，或指定软件名升级单个

卸载软件

winget uninstall chrome

彻底卸载，比控制面板更干净

查看详情

winget show vscode

显示软件版本、官网、描述等详细信息

实战场景 · 复制即用

1

新电脑装机必备（批量安装）

# 一行命令装5个常用软件
winget install Microsoft.VisualStudioCode Git.Git Mozilla.Firefox 7zip.7zip VideoLAN.VLC --silent
2

查找特定软件（精确ID）

# 搜索微信，找到准确的包名
winget search wechat

# 安装时使用精确ID避免装错
winget install Tencent.WeChat
3

导出/导入软件清单（换机神器）

# 旧电脑导出所有已安装软件
winget export -o "D:\apps.json"

# 新电脑一键恢复所有软件
winget import -i "D:\apps.json"

小白避坑指南

⚠️ 权限问题

部分软件安装需要管理员权限。如遇报错，右键"终端"选择"以管理员身份运行"再执行命令。

🔍 找不到软件？

不是所有软件都在 winget 仓库。搜索不到时，仍需手动下载安装包。

✅ 安装确认

默认安装前会提示确认（ Agreements ），按 Y 继续，或加 --accept-package-agreements 自动同意。

📦 多版本选择

搜索后注意看 Source 列：msstore 是微软商店版，winget 是官方源，ID 格式通常是 Publisher.AppName。
