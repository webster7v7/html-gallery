Windows 11 Starship 指南

3 分钟完成，禁止废话

步骤 1：安装
winget install --id Starship.Starship
步骤 2：配置 PowerShell
notepad $PROFILE

粘贴以下内容保存：

Invoke-Expression (&starship init powershell)
步骤 3：创建配置目录
mkdir "$env:USERPROFILE\.config"
步骤 4：应用主题（选一）
Pastel Powerline
Tokyo Night
Gruvbox Rainbow
步骤 5：安装字体（必须）

图标显示为方框？安装 Nerd Font：

下载 FiraCode Nerd Font →

解压 → 右键安装 → Windows Terminal 设置 → 选择 FiraCode Nerd Font

配置文件位置 %USERPROFILE%\.config\starship.toml

手动编辑配置示例：

# 基础配置
add_newline = true

[character]
success_symbol = "[➜](bold green)"
error_symbol = "[✗](bold red)"

[directory]
truncation_length = 3
style = "bold blue"

[git_branch]
symbol = "🌱 "
style = "bold purple"

常见问题

图标显示为方框？ 点击展开
未安装 Nerd Font 或终端字体设置错误。Windows Terminal 设置 → 配置文件 → 默认值 → 外观 → 字体 → 选择 FiraCode Nerd Font
提示无法加载配置文件？ 点击展开
执行策略限制。以管理员运行 PowerShell：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
$PROFILE 文件不存在？ 点击展开
if (!(Test-Path $PROFILE)) { New-Item -ItemType File -Path $PROFILE -Force }
