3x-ui

基于 Xray 核心的可视化代理管理面板

服务端 · 自建节点 · 流量管理

核心功能

🔌
多协议支持
VMess、VLESS、Trojan、Shadowsocks、Socks、HTTP 等主流协议一键配置
🌐
Web 管理面板
浏览器可视化操作，无需命令行，轻松添加删除用户、调整配置
📊
流量统计
实时监控每个用户流量使用情况，支持设置流量上限和到期时间
👥
多用户管理
一个端口可创建多个 UUID/密码，适合个人多设备或小型共享
🔒
自动证书
内置 Let's Encrypt SSL 证书自动申请和续期，保障连接安全
🔗
订阅链接
自动生成 Clash、v2rayN、Shadowrocket 等客户端订阅配置

角色定位对比

维度 3x-ui 服务端 Clash / v2rayN 客户端
运行位置 VPS / 云服务器（境外） 你的电脑 / 手机 / 路由器
核心作用 接收加密流量，解密转发到目标网站 把本地流量加密发送到服务端
使用人群 机场主、技术爱好者、自建节点用户 普通翻墙用户
技术门槛 需要 Linux 基础，购买境外 VPS 下载安装，导入订阅即可使用

工作流程

1
你的设备（Clash / v2rayN）
本地应用发起网络请求，通过客户端加密
↓
2
防火墙 / GFW
流量已加密，无法识别真实内容，被放行出境
↓
3
VPS 上的 3x-ui 面板 ← 你通过浏览器管理此处
接收加密流量，解密后转发到目标网站
↓
4
目标网站（Google / YouTube / GitHub）
收到请求，返回数据，原路加密传回你的设备

一键安装

在境外 VPS（Ubuntu/Debian/CentOS）上执行：

bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

安装完成后访问 http://你的服务器IP:2053 进入管理面板
默认账号密码请在安装完成后查看终端输出

3x-ui = 自己当机场老板
在境外 VPS 上部署，生成节点链接，供自己或他人使用
完全掌控流量、用户和隐私，无需依赖第三方服务
