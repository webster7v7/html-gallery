配置 Shell

初始化 PowerShell 配置

1

查看配置文件路径

echo $PROFILE
2

创建配置文件（如不存在）

New-Item -Path $PROFILE -Type File -Force
3

编辑配置文件，添加初始化代码

Invoke-Expression (& { (zoxide init powershell | Out-String) })

使用记事本或 VS Code 编辑： code $PROFILE

4

重新加载配置

. $PROFILE

Git Bash / MSYS2

eval "$(zoxide init bash)"

Command Prompt

CMD 支持有限，建议升级到 PowerShell 获得完整功能。

高级配置

自定义别名（使用 j 代替 z）

Invoke-Expression (& { (zoxide init powershell --cmd j | Out-String) })

排除特定目录

$env:_ZO_EXCLUDE_DIRS = "C:\Windows;C:\Program Files"

使用提示

首次使用需要积累目录访问历史，频繁使用几天后跳转会越来越精准。建议完全替代 cd 使用以获得最佳体验。 Zoxide 使用 Frecency 算法（频率 + 最近使用）对目录进行排序，越常用的目录匹配优先级越高。
