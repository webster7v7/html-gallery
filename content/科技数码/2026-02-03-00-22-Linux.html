内核发展时间线

Linux Kernel Evolution

1991

诞生

Linus Torvalds 在 comp.os.minix 新闻组发布著名帖子，宣布开发免费操作系统。

  • 最初使用 Minix 文件系统
  • 仅支持 386 处理器
  • 使用 GNU GPL 许可证前的过渡期
1992

GPL 许可证

Linux 内核采用 GPL 许可证，正式成为自由软件。Orest Zborowski 移植 X Window 系统。

1993

发行版元年

Debian、Slackware、Red Hat（前身）相继成立，Linux 发行版时代开启。

  • Slackware: 现存最古老的发行版
  • Debian: Ian Murdock 创立，强调自由软件
  • Red Hat: Marc Ewing 创立
1994

Linux 1.0

第一个正式版内核发布，支持单处理器 386 计算机，代码量约 176,250 行。

1998

企业关注

IBM、Oracle、Compaq 宣布支持 Linux。KDE 桌面环境发布，GIMP Toolkit (GTK+) 促进 GNOME 发展。

2004

Ubuntu 革命

Canonical 发布 Ubuntu 4.10 "Warty Warthog"，承诺每6个月发布新版本，彻底改变桌面 Linux 生态。

Mark Shuttleworth 资助，基于 Debian，强调易用性和"人类向所有人"

2007

移动时代

Google 发布 Android 系统，基于 Linux 内核，最终成为全球最流行的移动操作系统。

2016

WSL

微软发布 Windows Subsystem for Linux，Windows 与 Linux 的界限开始模糊。

最新 2024+

AI 与云原生

内核 6.x 系列支持 Rust 语言，AI 工作负载优化，eBPF 技术成熟，RISC-V 架构支持增强。

发行版家族谱系

基于包管理系统和继承关系的分类

.deb

Debian

1993 • Ian Murdock

完全由社区驱动，强调自由软件，以稳定性著称。拥有庞大的软件仓库（超过 59,000 个包）。

Ubuntu 最流行的桌面版
Linux Mint 对新手友好
Kali Linux 渗透测试
.rpm

Red Hat

1993 • Marc Ewing

企业级 Linux 的标杆，提供商业支持。RPM 包管理系统和 SELinux 安全增强。

RHEL 企业订阅
CentOS/Alma/Rocky RHEL 兼容
Fedora 新技术试验田
Rolling

Arch Linux

2002 • Judd Vinet

滚动更新模式，KISS 原则（Keep It Simple, Stupid），高度可定制，适合高级用户。

Manjaro 易用的 Arch
EndeavourOS 轻量安装器
ArcoLinux 学习平台
YaST

openSUSE

1994 • SUSE Software

德国工程血统，强大的 YaST 配置工具，提供 Leap（固定发布）和 Tumbleweed（滚动）双版本。

SLES 企业服务器
Leap 社区稳定版
Source

Gentoo

2000 • Daniel Robbins

源码编译安装，Portage 包管理，极致的性能优化控制，适合追求极致性能的用户。

Calculate Linux 预配置桌面
Pentoo 安全测试 LiveCD
TGZ

Slackware

1993 • Patrick Volkerding

现存最古老的活跃发行版，坚持 Unix 哲学，简洁至上，最接近原始 Unix 体验。

Salix 用户友好
VectorLinux 轻量快速
Functional

NixOS

2003 • Eelco Dolstra

声明式系统配置，可重现构建，原子升级回滚，革命性的 Nix 包管理器。

nix-darwin macOS 支持
Home Manager 用户配置
runit

Void Linux

2008 • Juan Romero Pardines

独立开发，使用 runit 替代 systemd，XBPS 包管理器，支持 musl libc 和 glibc 双版本。

musl 轻量 libc
Rolling Release 持续更新
musl/BusyBox

Alpine Linux

2005 • Alpine Team

面向安全应用，体积小巧（基础镜像仅 5MB），使用 musl 和 BusyBox，Docker 容器的首选基础镜像。

Docker Base 容器标准
Security PaX/SSP

Linux 趣闻

Tux 的名字

Linus Torvalds 曾被企鹅咬过，因此选择企鹅作为吉祥物。"Tux" 代表 Torvalds' Unix。

代码贡献

现代 Linux 内核代码量超过 3000 万行，但 Linus 只贡献了约 2% 的代码，主要做合并决策。

太空运行

国际空间站 (ISS) 于 2013 年将 Windows 笔记本更换为 Linux，以提高稳定性和安全性。
