1985

Windows 1.0

代号：Interface Manager

微软首个图形界面操作系统，引入鼠标支持、下拉菜单和窗口化应用。最初需要DOS支持运行。

  • 16位图形界面
  • 内置计算器、记事本、画图
  • 需要MS-DOS 2.0
  • 售价：$99
1990

Windows 3.0

首个商业成功的Windows

真正的转折点，引入程序管理器和文件管理器，支持虚拟内存和VGA图形，两年内售出1000万份。

  • 改进的内存管理
  • 支持386保护模式
  • TrueType字体
  • 纸牌游戏首次亮相
1995

Windows 95

开始菜单的诞生

革命性版本，引入开始菜单、任务栏和IE浏览器。发布时引起万人空巷的抢购热潮，定义了现代Windows界面。

  • 即插即用 (Plug and Play)
  • 32位操作系统
  • 长文件名支持
  • 内置TCP/IP协议
  • 启动音效由Brian Eno创作
2001

Windows XP

Luna 视觉风格

史上最长寿的Windows版本，标志性的蓝天白云壁纸和Luna主题。支持长达13年，是许多人心中的经典。

  • 全新的Luna视觉风格
  • 改进的启动速度
  • 远程桌面功能
  • 支持刻录CD
  • 2014年停止支持
2009

Windows 7

Vista的完美继任者

修复了Vista的问题，引入Aero Snap、改进的任务栏。被广泛认为是微软最成功的操作系统之一。

  • 库 (Libraries) 功能
  • 跳转列表 (Jump Lists)
  • 多点触控支持
  • HomeGroup网络共享
  • 2020年停止支持
2015

Windows 10

最后一个Windows版本？

回归开始菜单，引入Cortana、Microsoft Edge和Windows Hello。采用"Windows即服务"模式，持续功能更新。

  • 虚拟桌面
  • 通用Windows平台 (UWP)
  • WSL子系统
  • 游戏模式
  • 2025年10月停止支持
最新 2021-2025

Windows 11

现代设计与AI时代

全新居中的开始菜单，圆角设计，Android应用支持，以及Copilot AI助手。2025年9月发布25H2更新，持续进化。

  • 全新的Fluent Design
  • DirectStorage游戏加速
  • Snap Layouts窗口布局
  • Teams集成
  • Windows Copilot AI (24H2/25H2)
  • Wi-Fi 7 与 USB4 支持
